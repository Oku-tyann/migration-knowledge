## 情報7: 日本の出国税（国外転出時課税）の適用条件・対策

- 出典: https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1478.htm / https://mac-g.co.jp/international-tax/column/102/ / https://www.azn.co.jp/column/20220829-923.html
- 信頼度: 高
- 内容:
  - 制度開始: 2015年（平成27年）7月1日以後の国外転出が対象
  - 適用条件（2点とも満たす必要あり）:
    1. 資産要件: 国外転出時点で対象資産の時価合計が1億円以上
    2. 居住期間要件: 国外転出日前10年以内に合計5年超 日本国内に住所・居所があること
  - 対象資産: 有価証券等（株式・投資信託・公社債など）、匿名組合契約の出資の持分、未決済デリバティブ取引（不動産・預金・仮想通貨は現時点で対象外）
  - 課税の仕組み: 出国時に時価で全資産を売却したとみなすみなし譲渡課税
  - 税率: 所得税15%＋復興特別所得税0.315%＝15.315%
  - 申告期限: 納税管理人届出済みの場合→翌年3月15日まで（転出時の時価で申告）、届出なしの場合→出国時まで（出国3か月前の時価）
  - 納税猶予: 原則5年間（延長申請で最大10年間）、利子税が発生

## 情報8: 出国税の回避・軽減方法

- 出典: https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1478.htm / https://mac-g.co.jp/international-tax/column/102/
- 信頼度: 高
- 内容:
  - ①資産を1億円未満に抑える: 出国前に株式等を売却・現金化または贈与して時価合計を1億円未満にする（適用外）
  - ②居住期間要件を外す: 過去10年のうち日本居住が5年以下であれば適用外（海外長期滞在歴がある人）
  - ③5年以内に帰国する: 帰国時点で引き続き保有している資産については課税を取消し可能（最長10年）
  - ④帰国後に資産価格が下落した場合: 減額措置を受けられる
  - ⑤NISA口座内の資産は対象外（ただし非居住者になるとNISA口座は維持不可）

## 情報9: CFC税制（タックスヘイブン対策税制）とドバイ法人

- 出典: https://www.jetro.go.jp/world/qa/04A-010814.html / https://socpa.world/column/tax-haven-2025/ / https://socpa.world/column/taxhaven/ / https://www.agsc.co.jp/ags-media/2111tax_haven/
- 信頼度: 高
- 内容:
  - 外国関係会社の判定: 日本の居住者・内国法人が直接または間接に株式の50%超を保有している外国法人が対象
  - ドバイ（UAE）法人へのトリガー税率（令和5年度改正・2024年4月1日以後開始事業年度から）:
    - 特定外国関係会社（ペーパーカンパニー等）: 租税負担割合27%以上で適用免除
    - 一般の外国関係会社: 租税負担割合20%以上で適用免除
  - UAEの法人税率（2023年6月導入）は9%→いずれのラインも下回り原則としてCFC税制の検討対象
  - 経済活動基準（合算課税を免除する4要件）:
    1. 事業基準: 主たる事業が株式保有・知的財産の提供・船舶リース等でないこと
    2. 実体基準: 固定施設（事務所等）が現地にあること
    3. 管理支配基準: 事業の管理・支配が現地で行われていること（取締役会を現地開催等）
    4. 所在地国基準または非関連者基準
  - 個人株主への適用: 日本在住の個人が25%以上の株式を保有するドバイ法人は適用可能性あり
  - 本人が日本の非居住者になれば適用対象から外れる（非居住者化が前提）

## 情報11: タイ 海外所得課税 2024年改正の実態

- 出典: https://www.yamada-partners.jp/global-report/th-report-240305 / https://www.ht-tax.or.jp/topics/thai2024-mochikomikazei-rules/ / https://www.pwc.com/th/en/tax/assets/thai-tax/thai-tax-booklet-2024-25-jp.pdf
- 信頼度: 高
- 内容:
  - 居住者判定（183日ルール）: 暦年（1月1日〜12月31日）で180日以上タイに滞在した個人が税法上の「居住者」
  - 旧ルール（〜2023年12月31日）: 海外所得を稼得した同じ暦年内にタイへ持ち込んだ分のみ課税
  - 新ルール（2024年1月1日〜）: 稼得した暦年に関わらず、タイへ持ち込んだ暦年に課税
  - 経過措置: 2024年1月1日より前に稼得した所得は新ルールの対象外（2023年以前の貯蓄を2024年以降にタイへ送金しても課税なし）
  - 課税税率（累進課税）: 150,001〜300,000バーツ: 5% / 500,001〜750,000: 15% / 1,000,001〜2,000,000: 25% / 2,000,001〜5,000,000: 30% / 5,000,001〜: 35%
  - 日本・タイ租税条約あり（日本で課税済み所得についてはタイで外国税額控除が適用可能）

## 情報13: 日本住民票を抜く最適タイミング・手続き

- 出典: https://biz.moneyforward.com/establish/basic/81699/ / https://orchid-tax.com/tax-2/ / https://www.taxlawyer328.jp/inheritance-columns/海外移住/p7192/
- 信頼度: 高
- 内容:
  - 住民税の「1月1日ルール」: 住民税は1月1日時点で日本国内に住所がある人に課税
  - 12月31日以前に出国（住民票抜く）すれば翌年度の住民税は不要
  - 1月2日以降に出国すると当年1月1日時点で住民票あり→翌年度分の住民税が課税される
  - 最適な出国タイミング: 12月31日（実務的には12月中旬〜下旬が現実的）
  - 手続き: 海外転出届を出国予定日の14日前から当日までの間に市区町村役場へ提出
  - 必要書類: 本人確認書類（マイナンバーカード・運転免許証等）、印鑑、海外の住所が分かるもの
  - 注意点: 転出年月日の正確な記載が重要（届出日と転出日が異なる場合あり）

## 情報14: 住民票抜き後の注意事項

- 出典: https://mailmate.jp/ja/blog/kaigaitenshutsu-juminzei / https://5ftraveler.com/movingabroad/residencetax/ / https://www.taxlawyer328.jp/inheritance-columns/海外移住/p7192/
- 信頼度: 高
- 内容:
  - 国民健康保険: 転出届提出と同時に加入資格を失う（出国前に受診・医療を済ませる）
  - 国民年金: 非居住者になると任意加入（老齢年金維持したい場合は任意継続可）
  - 所得税の確定申告: 出国年の1月1日〜出国日までの所得について準確定申告が必要な場合あり
  - 納税管理人の届出: 出国後に日本で税務申告義務がある場合は税務署と市区町村に届出（義務）
  - 出国税との関係: 1億円以上の有価証券等を保有する場合は住民票を抜く前に出国税の申告・納税猶予手続きを完了
  - 帰国後は転入届を提出して住民票を再取得（帰国から14日以内）

## 情報17: オフショア法人（BVI・ジブラルタル）利用時の共通注意事項

- 内容:

  - 個人所得税: 居住者は0〜35%の累進課税（非居住者は域外所得に課税なし）
  - グローバルノマドビザ: 月€2,700以上の収入要件（月収入を証明）
  - 日本人向け注意: 税制が複雑で税理士費用が高く、コンプライアンスコストが大きい

## 重要警告・共通注意事項

- 日本に住民票を置いたまま海外法人を設立した場合、日本のCFC税制（外国子会社合算税制）が適用される可能性が高い
- 実際に節税効果を得るには「日本の非居住者」になることが前提（居住地移転必須）
- 出国税（国外転出時課税）: 有価証券等の時価合計が1億円以上の場合は出国前に要確認
- CRS（共通報告基準）により海外口座情報は自動的に日本の国税当局に報告される
- 必ず国際税務に精通した税理士・公認会計士に相談すること
