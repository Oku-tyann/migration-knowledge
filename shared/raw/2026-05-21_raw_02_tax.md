## 情報6: ジョージア 法人設立・スモールビジネス税制

- 出典: https://all-about.ge/tax_system/ / https://expathub.ge/tax-freelancers-individuals-small-businesses-georgia/ / https://ameblo.jp/gg-life/entry-12908508226.html
- 信頼度: 中
- 内容:
  - **法人税（LLC）**: 15%フラット税率（エストニア型：留保利益0%、分配時15%）
  - **バーチャルゾーン法人（IT企業向け）**: 法人税0%（ジョージア国外への収益が対象）
  - **個人所得税**: 20%フラット
  - **スモールビジネスステータス（1%税制）**:
    - 対象: 個人事業主（Individual Entrepreneur）
    - 税率: 年間売上の**1%**（フラット）
    - 売上上限: 500,000 GEL/年（約2,700万円相当）

    - 観光ビザで最大1年滞在可能（ビザなし）
    - 日本とジョージアの間に租税条約（DTA）なし→日本の非居住者申告義務に注意
    - 日本のCFC税制（タックスヘイブン対策税制）が適用される可能性あり
  - **VAT**: 18%（ジョージア国内向け）、国外向けサービスは免税
  - **ジョージアへの配当送金**: ジョージア国外居住の株主への配当は非課税

## 情報7: マルタ法人設立・実効税率5%の仕組み

- 出典: https://metiluxpartners.com/vol-1676 / https://www.la-quarta.jp/post/malta-tax-system / https://shiodome.co.jp/services/global-consulting/outbound/europe/malta/incorporation/
- 信頼度: 中
- 内容:
  - **基本法人税率**: 35%（EU内最高水準）
  - **税還付制度（Refund System）**:
    - 株主が配当を受け取る際、法人が納付した税金の**6/7**が還付される
    - 実効税率 = 35% × (1 - 6/7) = 約**5%**
    - 持株会社経由の場合は更に低減可能
  - **外国人による法人設立**:
    - 最低資本金: €1,135（有限会社）
    - 外国人100%株主・取締役が可能
  - **EU加盟国のメリット**: EU単一市場アクセス、英語が公用語
  - **適合ビジネス**: iGaming（オンラインカジノ）、ブロックチェーン、フィンテック、IT系サービス業
  - **注意点**: 
    - 税還付の手続きが複雑（専門の現地税理士が必須）
    - 実質的なマルタへの事業移転・実態が必要
    - EU規制（GDPR等）の遵守義務あり
  - **設立コスト目安**: 初期費用€3,000〜5,000（代行費含む）、年間維持€2,000〜5,000程度

## 情報9: タイ 海外所得課税 2024年改正・183日ルールの実態

- 出典: https://www.yamada-partners.jp/global-report/th-report-240305 / https://bangken.com/column/2024/0305-3041289/ / https://a-and-aa.com/news/2024/01/112/
- 信頼度: 高
- 内容:
  - **タイ居住者の定義**: 暦年（1月〜12月）内に**180日以上**タイに滞在した場合（≠183日。日本での「183日ルール」とは名称が異なる）
  - **2024年改正の核心**（2023年9月通達、2024年1月施行）:
    - **改正前**: 海外所得を同一課税年度内にタイへ持ち込んだ場合のみ課税
    - **改正後**: **2024年1月1日以降に発生した国外源泉所得**はタイへの送金年度にかかわらず、持ち込んだ時点で課税対象
    - 「年をまたいで送金すれば非課税」という従来の節税スキームが封じられた
  - **経過措置**:
    - 2023年12月31日以前に発生した所得 → 2024年以降にタイ送金しても非課税（発生時期が基準）
    - 2024年1月1日以降に発生した所得 → タイ持込年度を問わず課税
  - **タイ個人所得税率**: 累進課税、最高税率35%（課税所得5,000,000バーツ超）
  - **課税対象所得の具体例**:
    - 課税: 2024年以降に日本の株の配当・売却益を得てタイに送金した場合
    - 非課税: 2023年以前に蓄積した貯金をタイに送金した場合
  - **実務上の対策**:
    - 銀行口座の資金出所を明確に記録（いつ発生したか証明できる文書化）
    - タイ滞在日数を180日未満に調整（非居住者として課税回避）
    - LTR（Long-Term Resident）ビザ取得→海外所得が一定条件で免税

## 情報10: 日本 出国税（国外転出時課税）の条件・対策

- 出典: https://www.nta.go.jp/taxes/shiraberu/shinkoku/kokugai/01.htm / https://shiodome.co.jp/column/7730/ / https://mac-g.co.jp/international-tax/column/102/
- 信頼度: 高
- 内容:
  - **制度創設**: 平成27年（2015年）7月1日以降の国外転出から適用
  - **適用条件（2つとも満たす場合）**:
    1. 保有する対象資産の合計額が**1億円以上**
    2. 国外転出前10年以内に、**5年超**日本に住所または居所を有していた
  - **対象資産**:
    - 有価証券（株式・投資信託・公社債等）
    - 未決済信用取引・デリバティブ取引
    - NISA口座内の有価証券も含む
    - **不動産・現金・暗号資産は対象外**（注意：将来的な改正議論あり）
  - **課税内容**: 国外転出日時点の含み益に対して所得税・復興特別所得税が課税（15.315%または20.315%）
  - **主な対策・軽減方法**:
    1. **資産1億円未満に抑える**: 転出前に一部資産を売却して評価額を1億円未満にする
    2. **納税猶予制度**: 転出前に「納税管理人」の届出＋担保提供により、5年間（延長で最大10年間）の納税猶予が可能
    3. **帰国による取消し**: 猶予期間内に帰国（再び日本居住者になる）→課税取消し
    4. **贈与・相続への注意**: 出国後5年以内（延長で10年以内）に対象資産を国内居住者へ贈与・相続した場合も課税対象
  - **2024年改正**: 出国後の対象期間が5年から10年に延長（令和6年度改正）

## 情報11: CFC税制（タックスヘイブン対策税制・外国子会社合算税制）の適用条件

- 出典: https://www.mof.go.jp/tax_policy/summary/international/175.htm / https://koyano-cpa.gr.jp/nobiyo-kaikei/column/2849/ / https://maro-kaikei.co.jp/tax-haven-rules-japan/
- 信頼度: 高
- 内容:
  - **制度の目的**: 低税率国の外国子会社を利用した租税回避を防止し、その所得を日本親会社・居住者の所得に合算して課税
  - **適用対象**:
    - 内国法人（法人が外国子会社を保有する場合）
    - 居住者（個人が外国子会社を保有する場合）→ドバイ法人の個人オーナーも対象
  - **外国関係会社の判定**:
    - 日本の居住者・内国法人等が直接・間接に**50%超**の株式を保有する外国法人
  - **トリガー税率（適用免除基準）**:
    - 外国法人の租税負担割合が**27%以下**の場合、原則として合算課税の検討対象
    - **27%超**の場合、原則適用免除（ただし経済活動基準も別途必要）
  - **UAEドバイの税率とCFC適用**:
    - ドバイフリーゾーンのQFZP資格（実効税率0%）→ 租税負担割合が低いためCFC税制適用リスクあり
    - 法人税9%が適用される場合（27%未満）→ CFC税制の検討対象
    - **経済活動基準**を満たせば合算課税から除外可能：①事業基準（実質的な事業活動）②実体基準（固定施設あり）③管理支配基準（現地での意思決定）④非関連者基準または所在地国基準
  - **ペーパーカンパニー（適用除外なし）**:
    - 実体・管理支配基準のいずれかを満たさない → 問答無用で合算課税（パッシブ所得合算）
  - **適用税率**: 合算された所得に日本の税率（法人：最高約34%、個人：最高約55%）が適用される
  - **対策**: ドバイに実際にオフィス・スタッフを置き、現地での意思決定を証明できる体制を構築することが必須

## 情報12: 日本住民票を抜く最適なタイミングと住民税の関係

- 出典: https://5ftraveler.com/movingabroad/residencetax/ / https://biz.moneyforward.com/establish/basic/81699/ / https://www.umaumabali.com/post/resident-stay
- 信頼度: 高
- 内容:
  - **住民税の課税基準日**: **1月1日**（その年の1月1日時点で日本に住所がある者に課税）
  - **最適タイミング**:
    - 翌年度の住民税を完全回避するには、**前年12月31日までに出国し、かつ海外転出届（国外転出届）を提出済み**であることが必要
    - 例：2025年の住民税を回避するには2024年12月31日以前に転出届を出して出国
    - たった1日のズレ（1月1日に日本に住所あり）で、その年度の住民税が丸々課税される
  - **海外転出届（国外転出届）の提出義務**:
    - 海外に**おおむね1年以上**居住する場合、市区町村への転出届が義務（提出しない場合は5万円以下の過料）
    - 出国14日前から提出可能（市区町村役場で手続き）
  - **税務上の非居住者認定**:
    - 転出届提出後でも、「生活の本拠」が日本にあるとみなされた場合は居住者扱いのリスク
    - 条件: 家族・資産・ビジネスの中心が日本にある場合は要注意
  - **住民票を抜いた後の影響**:
    - 国民健康保険・国民年金の加入資格喪失
    - マイナンバーカードの失効（帰国時に再取得が必要）
    - 住民税の翌年度課税は発生しない（転出年度の前年所得に基づく住民税は転出年度に納付が必要）
  - **帰国時の注意**:
    - **1月2日以降**に帰国・転入登録すれば、その年の住民税は課税されない
    - 1月1日時点で日本に転入届を出すと、その年度の住民税が丸々課税される

## 情報16: BVI・タックスヘイブン対策税制（個人編）

- 出典: https://www.agsc.co.jp/ags-media/2111tax_haven/ / https://legacy.ne.jp/knowledge/before/fuyuusou/744-gaikoku-kogaisha-gassan-zeisei-kaisei-saishin-jouhou/
- 信頼度: 高
- 内容:
  - **個人へのCFC税制（租税特別措置法第40条の4〜6）**:
    - 居住者（個人）が外国関係会社の株式を**10%以上**保有する場合にも適用
    - 外国法人の所得が合算されて個人の総合課税所得に加算される
  - **適用免除（合算課税されない）条件**:
    - 租税負担割合が**30%以上**（個人の場合は30%以上で適用除外。法人の場合は27%以上）
    - 経済活動基準（事業基準・実体基準・管理支配基準・非関連者基準）を全て満たす
  - **ペーパーカンパニー判定**:
    - 現地に主要な資産・人員・意思決定機能がない場合 → 合算課税
    - BVI・ケイマン等のオフショア法人のほとんどが「ペーパーカンパニー」扱いを受けるリスク
  - **2024年以降の動向**:
    - OECDグローバルミニマム税（GloBE/第2の柱）：大企業（連結売上€750M超）対象に最低税率15%を確保する枠組み→小規模ノマドへの直接影響は少ないが、将来的な規制強化の流れ
  - **実務上の対応策**:
    - 日本から完全に非居住者化してから外国法人を設立（日本の住民票抹消・生活拠点移転が大前提）
    - 設立した外国法人に実質的な従業員・事務所・意思決定機能を持たせる

## 情報18: ドバイ法人設立で日本のCFC税制を回避するための実務要件

- 出典: https://www.meti.go.jp/policy/external_economy/toshi/kokusaisozei/cfc/PDF/170822_cfc.pdf / https://www.pwc.com/jp/ja/knowledge/glossary/tax-haven.html
- 信頼度: 高
- 内容:
  - **経済活動基準を満たすための具体的な要件**:
    1. **事業基準**: 株式保有・知的財産保有等の受動的事業（パッシブ系）でないこと（能動的なビジネスを行うこと）
    2. **実体基準**: 本店所在地国（ドバイ）に事業に必要な固定施設（事務所・設備）があること
    3. **管理支配基準**: 本店所在地国（ドバイ）で事業の管理・支配・運営が自ら行われていること
    4. **非関連者基準（または所在地国基準）**: 主要な取引先が関連会社でないこと、または所在地国での取引が主体であること
  - **フリーゾーン法人で経済活動基準を満たすポイント**:
    - フレックスデスクではなく、**専用オフィス**を借りる（実体基準の証明）
    - **現地スタッフを雇用**または代表者がドバイに居住・勤務する
    - 取締役会議事録・業務日誌等の記録を現地で作成・保管する
    - 銀行口座の取引がドバイ法人経由で適切に管理されていること
  - **最低ライン（よくある誤解）**:
    - 「フリーゾーン登録だけで節税完了」はNG
    - 日本に居住したまま（住民票あり）ドバイ法人を設立しても、CFC税制が適用されるだけでなく、出国税の問題も残る
    - まず日本の非居住者化（住民票抹消）が大前提
