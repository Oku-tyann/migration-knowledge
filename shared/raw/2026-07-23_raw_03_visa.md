## 1. タイ LTRビザ（Long-Term Resident Visa）概要

### 制度概要
- BOI（タイ投資委員会）が管理する最長10年の長期滞在ビザ
- 2022年導入、2025年改正で条件が大幅緩和
- 5年ビザ×2回更新（合計10年）の構造
- タイ国内申請時の10年マルチエントリービザ手続き料金: **50,000バーツ**

### 4つのカテゴリー

#### (1) Wealthy Global Citizens（富裕層）
- 条件: 世界資産100万USD以上 + タイ国内投資あり
- **2025年改正**: 「過去2年間の年収8万USD以上」という所得フロー要件が完全撤廃
- 所得証明の提出が不要になり、ストック資産重視の設計に変更
- 年齢制限なし
- タイ国内への海外送金所得: 免税優遇

#### (2) Wealthy Pensioners（富裕層退職者）
- 条件: 50歳以上
- 年間所得8万USD以上、または年間所得4万〜8万USD＋タイ国内投資25万USD以上（国債・直接投資・不動産）
- 10年居住許可付与

#### (3) Work-from-Thailand Professionals（リモートワーク専門職）
- タイ国外企業に雇用されたリモートワーカー向け
- 所得条件あり（詳細はBOI公式サイト参照）

#### (4) Highly Skilled Professionals（高度技術者）
- タイ国内特定業種（科学技術・ヘルスケア・金融等）向け
- 個人所得税優遇税率17%が適用可能

### LTRビザの主な特典
- 最大10年滞在許可
- デジタル労働許可証（ワークパーミット代替）
- 個人所得税優遇（海外所得免税 / 高度人材は17%フラット税率）
- 90日レポート: 年1回に緩和
- 配偶者・子の帯同可能

**情報源:** [タイ LTRビザ解説（リノシー）](https://www.dlife.co.jp/blog/longtermvisa-thai/) / [LTR条件緩和詳細](https://www.investorvisa.jp/blog/13564/) / [LTR Wealthy Global Citizens解説](https://thailand-ijyunavi.com/ltr-wealthy-global-citizens/)

## 2. タイランドエリート（Thailand Privilege Card）2026年版

### 概要
- タイ政府公認の長期滞在メンバーシップ制度
- 複数回入国ビザ + VIP空港サービス（スワンナプーム・ドンムアン）が付帯
- 2026年現在、5段階のプランが存在
- 代理店経由でも公式と同価格（価格は固定）

### プラン別料金（2026年）

| プラン | 年数 | 料金 | 1年あたり | 備考 |
|--------|------|------|----------|------|
| Bronze | 5年 | 650,000バーツ（約280万円） | 130,000バーツ | 2026年9月30日廃止予定 |
| Gold | 10年 | 900,000バーツ（約390万円） | 90,000バーツ | Bronze廃止後の最安エントリー |
| Platinum | 10年 | 1,500,000バーツ（約645万円） | 150,000バーツ | |
| Diamond | 15年 | 2,500,000バーツ（約1,075万円） | 167,000バーツ | 最多特典 |
| Reserve | 20年 | 5,000,000バーツ（約2,150万円） | 250,000バーツ | 招待制 |

### 重要な時期情報
- **Bronze廃止（2026年9月30日）**: 廃止後はエントリープランがGoldになり料金が250,000バーツ値上がり
- 現行価格は2026年中ロック済み

### LTRビザとの比較
- Thailand Privilege: 収入・資産証明が不要（メンバーシップ費用のみで取得可）
- LTRビザ: 資産・所得要件ありだが費用は安く税制優遇も強力
- 目安: 資産証明が難しいor手続きを簡素化したい場合はPrivilege、税制優遇を重視する場合はLTR

## 3. ドバイ（UAE）居住ビザ

### フリーランサー向けビザオプション（2026年）

#### (A) リモートワークビザ / デジタルノマドビザ
- 費用: AED 1,535（約6.4万円）〜AED 5,500（約23万円）
- 対象: UAE国外の雇用主のためにリモート勤務する個人
- 最低月収: USD 3,500〜5,000（約52万〜75万円）
- 有効期間: 1年（更新可）
- 2025年11月: GDRFA（ドバイ入国管理局）が書類審査を厳格化、クライアント契約書・安定収入の証拠が追加要求

#### (B) フリーランスビザ（フリーゾーン経由）
| フリーゾーン | 初年度総費用 | 特徴 |
|------------|----------|------|
| SHAMS（GoFreelance） | AED 12,830〜（約54万円） | クリエイティブ・メディア系 |
| IFZA | 約55〜80万円 | 幅広い業種、最短3〜5営業日 |
| Meydan | 約74万円〜 | |
| DMCC | 約144〜184万円 | 金融・貴金属系に強い |

- 申請期間: フリーランスパーミット承認8〜10営業日 + ビザ発行7〜14営業日（合計3〜6週間）

#### (C) フリーゾーン法人設立経由

- 条件: 不動産100万AED以上投資 / 月収30,000AED以上の高度人材 等

## 4. バンコク 180日滞在・ビザランに関する正確な情報

### 現行のビザ免除滞在期間
- 日本国籍: ビザなし入国で**60日間**の滞在許可（2024年7月改正後）
- タイ国内入国管理局で手続き（1,900バーツ）すればさらに+30日延長可能
- ただし**2026年5月タイ政府が60日免除を廃止し30日に戻す決定**を承認済み（施行日要確認）

### よくある誤解と実態

#### 誤解: 「年間180日以内なら何度でも入国できる」
**実態:**
- タイに「年間180日以内」という固定の法的ルールは存在しない
- 繰り返しのビザランは年2回程度が安全とみられるが、明確な回数制限の法文はない
- **問題はビザランの繰り返しに対する取り締まり強化**（入国審査官の裁量で入国拒否可能）

### ビザラン入国拒否の主な要因（2025〜2026年）
1. 陸路出国後、2泊以内に再入国（**2泊ルール**: 2025年導入）
2. パスポートにビザなし長期滞在・オーバーステイ歴がある
3. 正当な滞在理由（仕事・観光の実態）が示せない

### オーバーステイの罰則（現行）
| 超過期間 | 罰則 |
|---------|------|
| 1日〜（国内自首） | 1日あたり500バーツの罰金（上限20,000バーツ） |
| 90日未満（出国時） | 1日500バーツ罰金 |
| 90日以上 | 刑事罰として扱われる重大違反 |
| 90日超（出国後） | 1年間入国禁止 |
| 1年超（出国後） | 3年間入国禁止 |
| 3年超（出国後） | 5年間入国禁止 |

### 長期合法滞在の手段（2026年時点）
- **DTVビザ（Destination Thailand Visa）**: 5年有効・1回180日（+延長180日）・費用10,000バーツ
- LTRビザ、退職ビザ（Non-OA/OX）、タイランドエリートカード

**情報源:** [タイ入国ルール 2026版](https://enjoy-bkk.com/thailand-entry-rules-2026/) / [タイ観光ビザガイド 2026](https://www.starkitchen-thailand-gift.com/post/thailand-visa) / [ビザラン失敗実録](https://b-tabi.com/separate_wtravel/30thailandentryrefusal.html)

## 7. 補足: タイ DTV（Destination Thailand Visa）

- 5年有効のマルチエントリービザ
- 1回の入国で最大180日滞在（+延長申請で+180日）
- 費用: **10,000バーツ**（約4.3万円）
- 対象: リモートワーカー・フリーランサー・投資家・医療ツーリズムなど幅広い
- LTRビザやタイランドエリートより収入/資産要件の敷居が低い
- ビザランの代替として2024年後半から急増加
