## 1. タイ LTRビザ（Long-Term Resident Visa）

### 1-1. LTRビザ概要・申請プロセス
- **情報源**: https://ltr.boi.go.th/ / https://asean-jp.info/thailand/thailand-ltr-visa-definitive-guide-2026/ / https://www.siam-legal.com/ltr-visa-thailand/
- **概要**: タイBOI（投資委員会）が2022年9月から受付開始した10年間マルチプルエントリービザ。富裕層・退職者・リモートワーカー・高度人材の4カテゴリーがある。
- **申請ステップ**:
  1. BOIオンラインシステムで登録・書類提出（DOC.NO取得）
  2. 資格審査：書類受理後20営業日以内に結果通知
  3. 事前承認（Pre-approval）：1〜3営業日
  4. ビザ発給：承認通知書発行後60日以内に在外タイ大使館またはバンコクTIESCワンストップサービスセンターで申請
- **費用**: 50,000バーツ（タイ国内申請）
- **期間/更新**: 10年間有効（5年ごとに要件充足確認・更新手続き）
- **主な特典**:
  - 90日レポートが年1回に簡素化
  - 再入国許可不要
  - スワンナプーム・ドンムアン・プーケット・チェンマイ空港でのファストトラック
  - 配偶者および20歳以下の子ども最大4名まで連帯発給可
  - 高度人材向け：個人所得税17%優遇フラットレート
  - 海外所得は課税対象外（タイ非居住期間に生じた所得）

### 1-2. LTRビザ更新・年次報告
- **情報源**: https://asean-jp.info/thailand/thailand-ltr-visa-definitive-guide-2026/ / https://kyujin.careerlink.asia/blog/thailand-visa-extension-renewal-guide-2026/
- **更新要件確認**: 投資額維持・雇用継続・銀行残高10万USD以上を確認
- **申請窓口**: TIESCまたはBOI経由で申請（処理期間2〜4週間）
- **推奨タイミング**: 5年満了の3〜6ヶ月前に延長申請を開始
- **TIESCワンストップサービス（2025年3月開設）**:
  - 所在地: One Bangkok内Paradeゾーン6階（MRTルンピニ駅直結）
  - ラーマ4世通り側「Parade」入口→エレベーターで6階
- **初回報告義務**: LTR登録から1ヶ月以内にイミグレーション・ビューロー（またはTIESC）へ年次滞在報告（住所・雇用状況・家族構成等）

### 1-3. LTRカテゴリー① Wealthy Global Citizens（富裕層グローバル市民）
- **情報源**: https://asean-jp.info/thailand/thailand-ltr-visa-definitive-guide-2026/ / https://zero-asia.biz/information/visa_07.html / https://thailand-ijyunavi.com/ltr-wealthy-global-citizens/
- **概要**: 世界的な資産家・富裕層向け。2026年現在、要件は以下の通り。
- **要件（2026年版）**:
  - 個人名義の世界資産合計 **100万USD以上**
  - タイ国内への投資（国債・直接投資・不動産） **50万USD以上**
  - 医療保険50,000USD以上、または銀行預金100,000USD以上
  - ※「過去2年間の年収8万USD以上」の要件は完全撤廃（大幅緩和）
- **費用**: 50,000バーツ
- **期間/更新**: 10年（5年ごと更新）

### 1-4. LTRカテゴリー② Wealthy Pensioners（富裕層退職者）
- **情報源**: https://www.globalcitizensolutions.com/long-term-resident-visa-thailand/ / https://www.hlbthai.com/thailand-long-term-resident-ltr-visa/
- **概要**: 50歳以上のリタイア向け長期滞在ビザ
- **要件**:
  - 年齢 **50歳以上**
  - 年間所得 **80,000USD以上**、または
  - 年間所得40,000〜80,000USDの場合：タイ国内に **25万USD以上**投資（国債・不動産・直接投資）
  - 社会保障給付・年金収入も収入として算入可
- **費用**: 50,000バーツ
- **期間/更新**: 10年

### 1-5. LTRカテゴリー③ Work-from-Thailand Professionals（リモートワーカー）
- **情報源**: https://www.hlbthai.com/thailand-long-term-resident-ltr-visa/ / https://www.siam-legal.com/ltr-visa-thailand/
- **概要**: 海外企業にリモート勤務する専門職向け
- **要件**:
  - 年間所得 **80,000USD以上**（過去2年間）
  - 雇用先が上場企業、または設立3年以上かつ売上高 **1億5,000万USD以上** の企業
- **費用**: 50,000バーツ
- **期間/更新**: 10年

### 1-6. LTRカテゴリー④ Highly Skilled Professionals（高度専門人材）
- **情報源**: https://asean-jp.info/thailand/thailand-ltr-visa-definitive-guide-2026/ / https://a-and-aa.com/service/ltr/
- **概要**: タイの指定産業に従事する専門職向け
- **対象産業**: 次世代自動車・スマート電子機器・デジタル技術・医療・農業・食品・観光・教育・持続可能性エネルギー・バイオ技術・防災
- **要件**:
  - 年間所得 **80,000USD以上**（通常）、または
  - 修士号保持または専門知識所有の場合 **40,000USD以上**
- **費用**: 50,000バーツ
- **特典**: 個人所得税17%優遇適用

## 3. バンコク・タイ滞在180日ルールとオーバーステイ

### 3-1. 年間180日ルールの実態と誤解
- **情報源**: https://www.dlife.co.jp/blog/thailand-visa-newrules/ / https://pattayalife.net/archives/74732 / https://kyujin.careerlink.asia/blog/thailand-visa-extension-renewal-guide-2026/
- **正確な理解**:
  - 「180日ルール」とは法律上の明文規定ではなく、入国審査官の裁量による運用ガイドライン
  - タイは観光目的滞在を「年間180日程度」を目安に審査する非公式基準
  - 180日を超えた場合、続けての観光ビザ申請が却下されるケースが多い
  - 安全圏は年間150日以下が推奨
- **2026年5月の重要変更**:
  - 内閣が60日間ビザ免除措置の廃止を承認（官報掲載後15日で施行予定。掲載日は未定）
  - 廃止後は従来の30日ビザ免除に戻る見込み
- **ビザランの制限（2025年11月発表）**:
  - ビザラン上限：年2回まで
  - 正当な理由なく3回以上繰り返した場合、空港・国境で入国拒否のリスク
  - 2025年初め以降、入国拒否となった外国人は約2,900人以上

### 3-2. オーバーステイの罰則
- **情報源**: https://thailandelite.info/overstay/ / https://chiangmai43.com/overstay/ / https://www.juslaws.com/criminal-litigation/overstay-thailand-blacklist
- **罰金**: 1日500バーツ（上限20,000バーツ）
- **入国禁止期間（出国後）**:

| オーバーステイ期間 | 出国後の入国禁止期間 |
|-----------------|-----------------|
| 90日以内 | 罰金のみ（入国禁止なし） |
| 90日超〜1年未満 | 1年間 |
| 1年超〜3年未満 | 3年間 |
| 3年超 | 5年間 |

- **推奨対応**: 気づいた場合は入国管理局への自主出頭が最善

## 4. ドバイ（UAE）居住ビザ

### 4-1. フリーランサー向けビザ
- **情報源**: https://wise.com/jp/blog/freelance-visa-dubai / https://dubai-kenichi.com/freelance-visa/ / https://reinvent.co.jp/dubai-ijuu/
- **主な選択肢**:

| ビザ種類 | 特徴 | 収入要件 | 費用目安 |
|---------|------|---------|---------|
| フリーランスビザ | フリーゾーン経由で取得、1〜2年有効 | なし（業種制限あり） | AED 15,000〜30,000/年（62〜124万円） |
| グリーンビザ | 自営業・フリーランス向け長期（5年） | 年収360,000AED（約1,455万円）以上 | — |

- **フリーランスビザ対象職種**: WEBデザイン・映像・プログラマー・教師・経営コンサルティング・フィットネスコーチ等
- **取得ルート**: フリーゾーン運営機関経由 または 本土（メインランド）政府機関経由

### 4-2. フリーゾーン法人設立経由のビザ
- **情報源**: https://astravista-property.com/guide/company-setup / https://nexpert-law.com/web3/dubai-company-establishment-immigration / https://getfr33.com/dubai-free-zone/
- **概要**: フリーゾーンで法人設立し、スポンサービザとして居住ビザを取得する方法
- **メリット**: 外国人100%所有可能、法人税免税（フリーゾーン内取引）
- **初期費用**:
  - ビジネスライセンス料: AED 5,000〜25,000（約20万〜100万円）
  - ビザ費用: AED 3,000〜6,000/人（約12万〜24万円）
  - 初年度合計目安: **90万〜160万円**
- **年間維持費**: 80万〜120万円/年（翌年以降）
- **設立期間**: 書類が揃えば最短1週間でライセンス登録可能（通常は1ヶ月程度）
- **手順**:
  1. 会社名予約申請・書類準備
  2. フリーゾーンへのトレードライセンス申請
  3. 法人登記完了後に法人口座開設
  4. 居住ビザ申請（2〜10年有効、家族・社員ビザも取得可能）
- **2026年税務注意点**: 2023年6月導入の法人税9%が継続。売上年間400万AED（約1.2億円）以下の免税措置は2025年末で廃止予定、2026年以降は廃止見込み

## 5. 各国ノマドビザ比較

### 5-1. マレーシア DE Rantau（デ・ランタウ）
- **情報源**: https://www.mdec.my/derantau / https://klnomad.com/malaysia-digital-nomad-visa/ / https://mishu.my/blog/visa/de-rantau-application/
- **概要**: マレーシア・デジタル経済公社（MDEC）が2022年に開始したデジタルノマド向けプロフェッショナル訪問パス
- **収入要件**:
  - IT・デジタル専門職: 年間 **24,000USD以上**
  - 非IT職: 年間 **60,000USD以上**
- **主な条件**:
  - 18歳以上
  - マレーシア企業への就労禁止（海外収入源のみ可）
  - 健康保険加入必須
  - イスラエル国籍は申請不可
- **期間**: 12ヶ月（最大24ヶ月まで更新可能。満了3ヶ月前に申請推奨）
- **費用**: 約RM1,440以上（約315USD以上）。2025年5月1日より全手数料は審査結果に関わらず返金不可
- **処理期間**: 6〜8週間
- **地理的制限**: 半島マレーシアのみ有効。サバ州・サラワク州は別途観光ビザが必要
- **必要書類**: 申請書、有効なパスポート、証明写真、履歴書、雇用契約書または業務委託契約書、収入証明、健康保険証明

### 5-2. ポルトガル D8ビザ（デジタルノマドビザ）
- **情報源**: https://lottalingo.com/ja/research/visas/portugal-d8-digital-nomad / https://rikiyaishizaki.com/portugal-d8-visa/d8-visa-requirements / https://wise.com/jp/blog/digital-nomad-visa-portugal
- **概要**: EU圏でのデジタルノマド向け合法滞在ビザ
- **収入要件**: ポルトガル最低賃金の4倍以上
  - 2025年時点: 月額約3,040ユーロ以上
  - 2026年時点: 月額約3,680ユーロ以上（最低賃金改定による）
- **対象者**: 海外企業雇用のリモートワーカー・複数クライアントのフリーランサー・起業家
- **ビザ種類**:
  - 一時滞在ビザ（Temporary Stay Visa）: 最大1年（複数回更新可能）、申請料75ユーロ
  - 長期居住ビザ（Residence Visa）: 初回4ヶ月→2年居住許可取得→さらに3年更新（最大5年程度）、申請料90ユーロ
- **処理期間**: 最大90日
- **注意点**: 非EU国籍の方はシェンゲン圏外のポルトガル領事館で申請

### 5-3. インドネシア E33Gリモートワーカービザ（デジタルノマドビザ）
- **情報源**: https://rumavi.com/en/property-guides/bali-digital-nomad-visa-2026-e33g-remote-worker-guide / https://www.balivisas.com/digital-nomad-visa-bali-e33g/ / https://citizenremote.com/visas/bali-indonesia-digital-nomad-visa/
- **概要**: 2024年4月開始のフルタイムリモート従業員向けKITAS（長期滞在許可）
- **収入要件**: 海外収入源から年間 **60,000USD以上**、銀行残高 **2,000USD以上**（3ヶ月連続）
- **雇用条件**: インドネシア国外の企業と雇用契約必須（フリーランスは対象外）
- **パスポート条件**: 18ヶ月以上の有効期限、3ページ以上の白紙ページ
- **費用**: 自己申請 約600〜700USD、代行利用 約1,100〜1,600USD
- **期間**: 1年間（更新可能）
- **申請不可国**: アフガニスタン・カメルーン・ギニア・イスラエル・コソボ・リベリア・ナイジェリア・北朝鮮・ソマリア
- **Second Homeビザ（別途）**: 5年または10年滞在、預託金20億ルピア（約15万USD）以上必要

### 5-4. フィリピン デジタルノマドビザ（DNV）& SRRV

### 5-5. ジョージア デジタルノマド
- **情報源**: https://legal.ge/en/news/georgia-tightens-controls-as-digital-nomad-era-faces-regulatory-shift-en / https://citizenremote.com/visas/georgia-digital-nomad-visa/ / https://nomadvisa.me/georgia/
- **概要**: ジョージアは旧来、多くの国籍に対してビザなし1年滞在を許可していた（独立した「ノマドビザ」制度は持たない）
- **従来の条件（ビザなし滞在）**:
  - 月収 **2,000USD以上**、または銀行残高 **24,000USD以上**
  - 健康保険加入必須
  - 海外企業・クライアントへのリモートワーク
  - 申請費用: **無料**
- **処理期間**: 10〜30営業日
- **2026年の重要変更**:
  - 2025年6月26日、就労許可を義務化する法改正が可決
  - **2026年3月1日より外国人は就労許可が必要**に（施行済み）
  - ただし「ジョージア国内に入らない完全リモート（海外クライアント）」は許可不要の可能性あり（解釈確認中）
  - 長期滞在を考慮する場合は最新の法解釈を現地弁護士に確認すること

## 7. 補足情報・注意点

### 7-1. 最新トレンドと注意事項
- **タイの観光ビザ免除縮小（2026年5月）**: 内閣が60日間ビザ免除廃止を承認。30日免除に戻る見込み。施行日は官報掲載後15日（未定）
- **ドバイの法人税**: 2023年6月導入の9%法人税。フリーゾーン優遇措置の一部が2026年以降廃止予定
- **マレーシア DE Rantauの地理的制限**: 半島マレーシアのみ有効。ボルネオ島（サバ・サラワク）訪問時は別途ビザ必要
