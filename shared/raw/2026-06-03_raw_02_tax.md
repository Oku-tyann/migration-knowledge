# 税制・法人調査 — 2026-06-03

## 情報1: ドバイ（UAE）フリーゾーン法人税制の全体像

- 出典: https://www.jetro.go.jp/world/middle_east/ae/invest_04.html / https://dubai-business-consulting.com/corporate-tax/
- 信頼度: 高
- 内容:
  - UAE法人税は2023年6月1日以降の事業年度から課税所得に対し **9%** 適用
  - 個人所得税・住民税は **完全無税**（個人レベルでの課税ゼロ）
  - フリーゾーン法人は「Qualifying Free Zone Person（QFZP）」として認定されれば **適格所得（Qualifying Income）は0%** 免税
  - QFZP認定の条件：適格活動（Qualifying Activities）以外からの売上が総売上の **5%またはAED500万以下**
  - 重要な落とし穴：**自然人（個人）との直接取引** は原則として非適格所得扱いになるため、B2C型オンラインビジネスはQFZP認定が困難な場合あり
  - 課税所得 **AED375,000（約1,500万円）まで0%**、超過分に9%
  - グローバルミニマム課税（Pillar 2・15%）の対象は年間総収入 **€7.5億以上** の多国籍企業のみ → 中小個人事業者は対象外
  - UAEはDMTT（国内補完税）を2025年1月から導入したが大企業のみ対象

## 情報6: マルタ法人の税率リファンド制度詳細

- 信頼度: 中
- 内容:
  - 名目法人税率：**35%**（EU最高水準のひとつ）

  - リファンド受取には専門会計事務所の利用が必須（費用：年間€3,000〜6,000程度）
  - EU加盟国であるため、**日本のCFC判定基準（実体要件）を満たすかが重要課題**

## 情報9: 日本の出国税（国外転出時課税）の発動条件と対策

- 出典: https://www.nta.go.jp/taxes/shiraberu/shinkoku/kokugai/01.htm / https://mac-g.co.jp/international-tax/column/102/ / https://tomorrowstax.com/knowledge/2025012013979/
- 信頼度: 高
- 内容:
  - 制度概要：2015年7月1日以降の国外転出（海外移住）時に発動する「みなし譲渡課税」
  - **発動条件**：以下を同時に満たす場合
    1. 対象資産の合計額が **1億円以上**
    2. 過去10年間のうち **5年超、日本に住所または居所があった** 居住者
  - **課税対象資産**：
    - 有価証券（株式・投資信託等）
    - 匿名組合契約の出資持分
    - 未決済の信用取引・発行日取引・デリバティブ取引
    - ※不動産・現金・暗号資産（仮想通貨）は **対象外**
  - **課税タイミング（1億円の判定日）**：
    - 出国前に申告する場合：出国予定日の **3ヶ月前の日**
    - 納税管理人を選任して出国後に申告する場合：**出国日当日**
  - **納税猶予制度**：
    - 納税管理人の届出のみ：出国後 **5年間** 猶予
    - 納税管理人の届出 + 担保提供：最長 **10年間** 猶予
    - 猶予期間内に帰国すれば課税は取消・消滅
  - **主な対策**：
    - 有価証券を1億円未満に圧縮してから出国（売却・贈与等）
    - 不動産や現金等の非対象資産にシフト
    - 暗号資産（仮想通貨）は対象外なので保有形態の見直し
    - タイミング管理（3ヶ月前判定の活用）
    - ただし税回避目的の操作には国税当局が注目

## 情報10: 日本のCFC税制（タックスヘイブン対策税制）とドバイ法人への適用

- 出典: https://www.jetro.go.jp/world/qa/04A-010814.html / https://socpa.world/column/tax-haven-2025/ / https://www.agsc.co.jp/ags-media/2111tax_haven/
- 信頼度: 高
- 内容:
  - CFC税制の仕組み：日本居住者が実質的に支配する外国法人の所得を、日本親会社の所得に **合算して課税**
  - **トリガー税率**：かつて25%、現行制度では「租税負担割合 **20%未満**」が合算課税の対象
  - UAEの法人税率9%は20%を大きく下回る → ドバイ法人は **CFC判定対象**
  - **適用免除基準（事実上のトリガー税率代替）**：租税負担割合20%以上であれば自動免除
  - ドバイ法人が合算課税を免れる条件：**経済活動基準（実体要件）を満たすこと**
    - 主要な事業活動を外国（UAE）で行っている
    - 実質的な事業運営（人員・設備・リスク管理等）をUAEで行っている
  - **根本的な解決策**：日本の非居住者になれば、CFC税制は適用されない
  - 結論：ドバイ法人設立だけでは不十分。**日本住民票除票 + UAE居住者ビザ取得 + 実体ある事業運営** の三点セットが必要

## 情報11: 日本の住民票を抜く最適タイミングと注意点

- 出典: https://biz.moneyforward.com/establish/basic/81699/ / https://doda.jp/global/guide/kaigaiprocedure.html / https://tenshoku.mynavi.jp/global/z/procedure/
- 信頼度: 高
- 内容:
  - **住民税の判定日：毎年1月1日**（この日に日本に住民登録があれば、その年の住民税が課される）
  - **住民税を回避するための大原則**：前年の **12月31日までに海外転出届を提出し、住民票を除票する**
  - 例：2025年1月1日出国 → 2025年分の住民税（翌年6月まで払い続ける分）が発生してしまう
  - 例：2024年12月31日出国 + 転出届提出 → 2025年分の住民税は不要
  - **海外転出届の提出期限**：出国の **14日前から**（一部自治体は1ヶ月前から受付）
  - **住民票除票後の影響**：
    - 国民健康保険：資格喪失 → 出国後は加入不可
    - 国民年金：免除される（任意加入も可能）
    - 住民税：翌年分から不要
    - 扶養家族の保険：別途検討が必要
  - **国民年金の選択肢**：
    - 任意加入可能（海外在住中も保険料納付で老齢年金を維持）
    - 任意加入しない場合：その期間は欠期となり将来の受給額が減少
  - **マイナンバーカード**：海外転出後も一定条件で継続利用可能（2024年改正）

## 情報12: タイの海外所得課税2024年改正の実態

- 出典: https://www.yamada-partners.jp/global-report/th-report-240305 / https://bangken.com/column/2024/0305-3041289/ / https://a-and-aa.com/news/2024/01/112/
- 信頼度: 高
- 内容:
  - **改正前のルール**（〜2023年末）：海外所得をタイに送金した場合、**同年度内に稼得した分のみ** 課税。翌年以降に送金すれば非課税
  - **2024年1月1日からの新ルール**：タイ居住者（年間180日以上滞在）が海外源泉所得をタイに持ち込んだ場合、**稼得年度に関わらず** 持ち込んだ年度に個人所得税（最高35%）が課税
  - **経過措置**：2024年1月1日より前に稼得した所得は新ルールの対象外（2024年以降に送金しても非課税）
  - **居住者の定義**：タイに年間 **180日以上** 滞在（183日ではなく180日が正確）
  - **非居住者は影響なし**：180日未満の滞在者にはこの改正は適用されない
  - **二重課税救済**：租税条約（日タイ条約含む）に基づき、外国で納付済みの税金はタイの所得税から控除可能
  - **実務的影響**：
    - タイ長期滞在（LTRビザ・タイランドエリートビザ等）利用者に大きな影響
    - 節税目的でタイを選ぶ日本人にとっては想定外の課税リスク
    - ただし、タイ国内での執行体制・実際の申告義務の周知はまだ十分でないとの指摘あり
  - **タイ個人所得税率**：累進課税、最低5%〜最高35%（課税所得5,000万バーツ超）
