## 情報4: シンガポール法人税17%と各種優遇制度

- 出典: https://www.jetro.go.jp/world/asia/sg/invest_04.html / https://www.ey.com/ja_jp/technical/japan-business-services/jbs-insights/accounting-auditing-tax-guides/ey-japan-jbs-singapore-guide-2025 / https://sg-bizadvisor.com/2020/04/19/シンガポール法人税の課税対象所得は/
- 信頼度: 高
- 内容:
  - 法人税フラット税率17%。世界的に見ても低水準。
  - キャピタルゲイン税なし。配当の源泉税なし（シングルティア税制）。
  - 部分免税制度: 課税所得のうち最初のS$10,000の75%、次のS$290,000の50%が免税（恒久措置）。
  - **スタートアップ向け新設法人免税（Tax Exemption Scheme for New Start-up Companies）**: 設立後3賦課年度間、最初のS$100,000の75%免除、次のS$100,000の50%免除。
  - 適用条件: シンガポール設立法人、居住法人、株主20名以下、個人株主のうち少なくとも1名が10%以上保有。
  - 2025年の税制改正: 法人所得税で2,000SGDの現金支給＋S$40,000を上限に50%の税額控除。
  - テリトリアル課税（源泉地主義）が原則で、シンガポール外で発生した所得は原則非課税。

## 情報7: マルタ法人の実効税率5%の仕組み

- 出典: https://metiluxpartners.com/vol-1676：マルタで法人設立するメリットと税金の実態【35%課税でも実効税率5%？】 / https://www.la-quarta.jp/post/malta-tax-system
- 信頼度: 中
- 内容:
  - 名目法人税率は35%と高いが、税還付制度（Tax Refund System）により株主への還付後、**実効税率は約5%**。
  - 仕組み: 法人が35%を納税→株主がマルタ国外居住の場合、配当受取時に最大6/7（約30%相当）の還付を受けられる。
  - 最低資本金: 有限会社（Private Limited Company）設立には1,135ユーロ。
  - 相続税・キャピタルゲイン税・富裕税: 非課税。
  - 70以上の国と二重課税回避条約締結（日本との条約あり）。
  - EUメンバー国。ノマドビザも発給開始済み（月収要件約2,700ユーロ）。
  - 注意: 実際の還付手続きや会計費用が発生するため、実務コストを考慮する必要あり。

## 情報8: ジョージア（カフカス）の法人設立と税制

- 出典: https://all-about.ge/tax_system/ / https://georgia.journey-coordinator.com/georgia-small-business-status/ / https://offshore-houjin.com/taxsavingknowledge/georgiaoffshore/
- 信頼度: 中
- 内容:
  - 法人税: エストニア型の分配時課税方式（2017年導入）。留保利益への課税0%、配当分配時に15%。
  - **IT Virtual Zone（仮想ゾーン）**: 海外顧客へのIT/ソフトウェアサービスから得た収益は法人税0%。ただし事務所賃借・ジョージア人スタッフ雇用が条件。
  - **スモールビジネスステータス**: 個人事業主として登録し、年間売上500,000GEL（約2,700万円）未満かつ年間182日以上滞在で、売上の1%のみ課税。
  - 法人設立費用: 低コスト（100〜200GEL程度）。ただし近年、外国人の銀行口座開設はAML規制強化により難化。
  - 日本人は観光ビザ不要で1年間滞在可能（無条件）。
  - 注意: 完全なオフショア法人（実体なし）としての利用は難しい。AML規制のリスクあり。

## 情報9: BVI（英領バージン諸島）オフショア法人

- 出典: https://offshore22.com/bvi/ / https://getfr33.com/bvi-corporations/ / https://www.offshore-protection.com/offshore-blog/offshore-company-digital-nomads-perpetual-travelers
- 信頼度: 中
- 内容:
  - 外国源泉所得は完全免税。法人税・キャピタルゲイン税・相続税・源泉税なし。
  - 取締役・株主・受益者情報の公開義務なし（高い匿名性）。
  - 設立費用: 年間ライセンス費用約1,100〜1,500USD程度。
  - 設立期間: 3〜5営業日。最低取締役・株主1名（同一人物可、国籍制限なし）。最低資本金なし。
  - 世界最大のオフショア法人登録地。
  - 重大な注意点: 日本人が日本居住のまま設立した場合、**CFC税制（タックスヘイブン対策税制）の適用対象になり得る**。実体のある事業活動が必要。
  - 銀行口座開設が近年困難化（実体ある事業証明が求められる）。

## 情報10: タイ2024年改正——海外所得課税ルールの大幅変更

- 出典: https://www.yamada-partners.jp/global-report/th-report-240305 / https://www.ht-tax.or.jp/topics/thai2024-mochikomikazei-rules/ / https://pineconeblog.com/thailand-overseas-income-taxation/
- 信頼度: 高
- 内容:
  - **2023年9月に通達発布、2024年1月1日から適用**。
  - 旧ルール: 同一年度内にタイへ持ち込んだ海外所得のみ課税。翌年以降の持ち込みは非課税だった。
  - **新ルール**: 2024年1月1日以降に発生した海外所得は、タイへの持ち込みがいつであっても課税対象。年度をまたいで持ち込んでも課税される。
  - 経過措置: 2023年12月31日以前に発生した所得は引き続き非課税（タイに持ち込んでも問題なし）。
  - 対象: タイに**年間180日以上滞在するタイ居住者**（183日ではなく180日が正確な基準）。
  - 実態: 課税対象となる所得の種類・二重課税防止条約との関係など実務上の解釈が継続検討中。
  - 2024年秋頃にFAQが歳入局から発出され詳細が明確化。

## 情報11: 日本の出国税（国外転出時課税）の条件と対策

- 出典: https://www.nta.go.jp/taxes/shiraberu/shinkoku/kokugai/01.htm / https://mac-g.co.jp/international-tax/column/102/ / https://shiodome.co.jp/column/7730/
- 信頼度: 高
- 内容:
  - **対象者要件**:
    1. 国外転出（出国）する居住者
    2. 出国前10年間のうち5年超、日本に住所・居所を有していた者
    3. 保有する対象資産の合計額が1億円以上
  - **対象資産**: 有価証券（株式・投資信託等）、デリバティブ取引に係る権利、未決済信用取引等。
  - **課税対象**: 対象資産の含み益（未実現利益）に所得税・復興特別所得税が課税される。
  - **主な対策**:
    1. 出国前に対象資産の時価総額を1億円未満に下げる（贈与・売却等）
    2. **納税猶予制度**: 最大10年間の猶予。帰国予定ありの場合、担保提供で猶予申請可。猶予期間中に売却等がなければ原則課税なし。期間内に帰国すれば取消し可。
    3. 過去10年間の日本居住期間が5年以内になるよう移住前から対策する。

## 情報12: CFC税制（外国子会社合算税制・タックスヘイブン対策税制）の適用条件

- 出典: https://socpa.world/column/tax-haven-2025/ / https://maro-kaikei.co.jp/tax-haven-rules-japan/ / https://socpa.world/blog/taxhaven/
- 信頼度: 高
- 内容:
  - **適用対象（外国関係会社の要件）**: 日本居住者・内国法人が直接または間接に株式50%超を保有する外国法人、または実質支配関係がある外国法人。
  - **トリガー税率**: 租税負担割合が27%未満の場合にCFC税制の検討が必要（改正後の基準）。ドバイ法人の実効税率9%はこれを下回る。
  - **適用除外（経済活動基準）**: 以下の全4要件を満たせば合算課税を免れる。
    1. 事業基準（主事業が株式保有・貸付・特許等でない）
    2. 実体基準（現地に固定施設あり）
    3. 管理支配基準（現地で事業の管理・支配・運営を自ら行う）
    4. 非関連者基準または所在地国基準
  - **重要**: 日本の個人株主が**日本の非居住者**になれば、日本のCFC税制の適用対象から外れる（主要な節税ポイント）。
  - ドバイ法人をCFC税制適用外にするには: ドバイに実際の事務所、スタッフ、経営実態を持たせることが必須。

## 情報13: 日本住民票を抜く最適タイミング（1月1日ルール）

- 出典: https://biz.moneyforward.com/establish/basic/81699/ / https://bs.taxlawyer328.jp/blog/moving-abroad/p12862/
- 信頼度: 高
- 内容:
  - **住民税の1月1日ルール**: 住民税は毎年1月1日時点の居住地に基づき課税される（前年の所得に対して翌年に請求）。
  - **翌年の住民税を完全回避するには**: 12月31日までに出国し、かつ海外転出届を市区町村に提出して住民票を除票しておく必要あり。
  - 例: 2025年12月31日までに出国・住民票除票 → 2026年の住民税なし。1月1日出国では2026年度の住民税が課される。
  - **海外転出届の提出時期**: 出国日のおおむね14日前〜出国当日までの間に役所窓口で手続き。
  - 重要: 株式・不動産売却等で大きな利益が発生する場合、移住タイミング（1か月差）で数十万〜数百万円の差が出る。
  - 1年未満の短期滞在・留学は短期滞在扱いで住民票除票の必要なし。

## 情報14: 非居住者になった後の日本所得税・二重課税

- 出典: https://wise.com/jp/blog/non-resident-income-tax / https://www.ht-tax.or.jp/sozoku-guide/overseas-migration-tax
- 信頼度: 高
- 内容:
  - 非居住者は日本国内源泉所得（日本不動産の賃料・日本株の配当等）のみ日本で課税。海外所得は日本では課税されない。
  - 非居住者の日本株配当: 源泉徴収20.315%（日本との租税条約により減免される場合あり）。
  - 非居住者でも日本国内に恒久的施設（PE）があれば事業所得に課税される。
  - 移住先国との二重課税は租税条約や外国税額控除で調整。
  - **出国後も日本の確定申告が必要なケース**: 国内源泉所得がある場合、出国年の所得税は帰国前に「納税管理人」を選任して申告。

## 調査メモ・総合所見

1. **ドバイ最大の変化**: 2023年6月の法人税導入で「ドバイ＝法人税ゼロ」の時代は終了。フリーゾーンは条件次第で0%維持可能だが、適格要件の確認が必須。
2. **タイ2024年改正の影響大**: 従来タイ居住＋海外所得は翌年持ち込みで非課税だったが、2024年以降に発生した所得は持ち込み年度に関係なく課税対象。タイ拠点のノマドは対策が急務。
3. **CFC税制の壁**: 日本居住のまま低税率国に法人を設立しても、CFC税制により日本で合算課税される。節税効果を得るには日本の非居住者になることが最も確実。
4. **住民票除票は12月31日まで**: 翌年の住民税完全回避には年内出国＋住民票除票が必須。タイミングを誤ると数十万〜数百万円の損失になりうる。
5. **出国税対策は早めに**: 金融資産が1億円を超える前に移住する、または移住の数年前から資産圧縮・分散を計画することが重要。
