## 情報3: ドバイ フリーゾーン法人 QFZP（0%税率）資格条件

- 出典: https://www.naviracorporate.com/blog/qualifying-income-uae-corporate-tax/ / https://kpmg.com/ae/en/insights/tax-insights/updated-rules-for-qualifying-free-zone-persons.html
- 信頼度: 高
- 内容:
  - UAE法人税法Article 18 + Ministerial Decision No. 229 of 2025（2025年8月28日発効）により条件が規定
  - **5つの累積条件（全て満たす必要あり）**:
    1. **実質要件（Adequate Substance）**: フリーゾーン内でコア活動を実施、十分な従業員・資産・運営費用が必要
    2. **適格所得（Qualifying Income）**: フリーゾーン内に物理的オフィス、有資格従業員、意思決定機能が必要
    3. **監査済み財務諸表**: 2023年6月1日以降の全税務期間で必須
    4. **移転価格コンプライアンス**: 関連者間取引はアームズレングス原則に準拠
    5. **標準課税制度への移行申請をしないこと**
  - **De Minimis閾値**: 非適格所得がAED 500万円 or 総収入の5%を超えるとQFZP資格失効 → 4年間のクーリングオフ
  - 資格失効時は当年度+翌4年間、全所得に9%課税
  - MD 229（2025年改正）で適格活動の範囲が拡大 → より多くのフリーゾーン事業が0%対象に
  - 実質のない「ペーパー法人」では条件を満たせず9%課税になる点に注意

## 情報5: タイの海外所得課税 2024年改正の全体像

- 出典: https://www.yamada-partners.jp/global-report/th-report-240305 / https://bangken.com/column/2024/0305-3041289/ / https://www.bdo-japan.jp/ja-jp/insights/publications/20241218/thailand
- 信頼度: 高
- 内容:
  - **タイ居住者の定義**: 暦年中のタイ滞在日数が180日以上（≠183日。タイは180日基準）
  - **改正前（〜2023年）**: 国外源泉所得は「稼得した同じ暦年内」にタイ持ち込みの場合のみ課税対象。翌年以降の持ち込みは非課税 → いわゆる「年越しテクニック」が有効だった
  - **改正後（2024年1月1日〜）**: 2024年以降に発生した国外源泉所得は、タイ持ち込み時期に関わらず全て課税対象に
  - **経過措置**: 2023年12月31日以前に稼得した所得は旧ルール適用継続 → 2023年以前の蓄積資産の持ち込みは非課税
  - **対象所得**: タイ国税法第40条（1）〜（8）の全所得種別（給与、事業所得、利子、配当、不動産収益等）
  - **実態**: 日本からタイに移住し海外口座の所得をタイ口座に送金すると課税リスク発生
  - **対策**: タイ国内で消費せず第三国（UAE等）口座に置く、または非課税口座・貯蓄からの持ち込みに整理する方法を検討
  - タイ・日本間には租税条約があり二重課税防止規定あり。ただし申告手続きが必要

## 情報6: タイ 183日ルールの誤解と実態（2025年時点）

- 出典: https://zero-asia.biz/information/income_tax_01.html / https://kaipro.link/articles/foreign-sourced-income-brought-into-thailand/
- 信頼度: 中
- 内容:
  - タイの居住者判定は「暦年180日以上の滞在」→「180日ルール」が正確（日本では慣用的に183日と言われることも）
  - 居住者認定されると全世界所得（タイ持ち込み分）が課税対象
  - 2024年改正後の追加論点: **持ち込み所得課税が対象年度を問わない**に変更。過去に稼得した所得もタイに持ち込めば課税（ただし2023年以前分は経過措置）
  - ノマド戦略への影響: 「タイ在住で海外口座の利益を日本で課税、タイには持ち込まない」戦略は引き続き有効だが送金管理が複雑化
  - 2025年現在: LTR（Long-Term Resident）ビザ保有者は一部免税特例あり（特定の外国所得が非課税）
  - タイ歳入局による実務執行は始まっており、特に外国人の確定申告提出増加傾向

## 情報7: 日本の出国税（国外転出時課税制度）詳細

- 出典: https://www.nta.go.jp/taxes/shiraberu/shinkoku/kokugai/01.htm / https://tomorrowstax.com/knowledge/2025012013979/ / https://mac-g.co.jp/international-tax/column/102/
- 信頼度: 高（国税庁公式）
- 内容:
  - **対象者**: 以下の両方を満たす海外転出者
    1. 出国時に時価合計**1億円以上**の対象資産を保有
    2. 転出前10年以内に**5年超**日本国内に居住
  - **対象資産**: 上場・非上場株式、投資信託、公社債、未決済デリバティブ取引等。**仮想通貨（暗号資産）は2025年7月時点で課税対象外**
  - **課税内容**: 含み益に所得税（15%）＋復興特別所得税（0.315%）= 約20.315%を出国時みなし譲渡で課税
  - **納税猶予制度**: 納税管理人を選任し担保提供すれば、国外転出から5年間（延長申請で最大10年間）猶予可能
  - **帰国時の取り消し**: 5年以内（延長時10年以内）に帰国し4ヶ月以内に申請すれば税額取り消し可能
  - **対策の方向性**:
    - 資産が1億円未満になるよう事前に売却・整理（含み益確定）
    - 贈与税非課税枠を活用した資産移転
    - 転出前5年以内の居住年数を調整（難易度高）
    - 暗号資産への一部移行（現状課税対象外だが将来改正リスクあり）
  - 国外転出届と確定申告を出国前に要実施

## 情報8: CFC税制（外国子会社合算税制）の適用条件とドバイ法人

- 出典: https://maro-kaikei.co.jp/tax-haven-rules-japan/ / https://www.agsc.co.jp/ags-media/2111tax_haven/ / https://www.jetro.go.jp/world/qa/04A-010814.html
- 信頼度: 高
- 内容:
  - **トリガー税率**: 外国子会社の実効税率が20%未満の場合にCFC税制の審査対象
  - **UAE（ドバイ）の法人税率**: 9% → 20%未満 → CFC税制の検討対象に該当
  - **適用ステップ**:
    1. 外国関係会社かどうか（日本居住者が直接・間接に50%超の株式等保有）
    2. 会社単位合算: 租税負担割合20%未満 → 経済活動基準4条件を全て満たすか確認
    3. 4条件: ①事業基準（主たる事業が非適格でない）②実体基準（固定施設保有）③管理支配基準（役員・従業員が現地で実際に管理）④所在地国基準 or 非関連者基準
    4. 4条件を1つでも満たさない → 全所得が日本の株主の所得に合算課税
  - **令和7年（2025年）改正**: 合算課税の時期が「2ヶ月」から「4ヶ月」に変更
  - **BVI・ケイマン等のオフショア法人**: 税率ほぼ0%、実体なし → CFC税制ほぼ確実に適用
  - **ドバイ法人でCFC税制を回避する主な方法**:
    - ①株主が日本の非居住者になる（最も確実）
    - ②実体のある事業活動をドバイで行い4条件を満たす（管理支配基準が特に重要）
  - 個人が外国法人株式100%保有の場合も適用あり（法人・個人問わず）

## 情報9: 日本住民票を抜く最適タイミングと実務手順

- 出典: https://bs.taxlawyer328.jp/blog/moving-abroad/p12862/ / https://5ftraveler.com/movingabroad/residencetax/ / https://biz.moneyforward.com/establish/basic/81699/
- 信頼度: 高
- 内容:
  - **住民税の1月1日ルール**: 翌年度の住民税は「1月1日時点の住民登録」で決まる
    - 例: 2025年12月31日までに転出届提出・除票 → 2026年分住民税は発生しない
    - 1日でも遅れると翌年分（数十〜数百万円）の住民税が発生するケースあり
  - **転出届の手続き**:
    - 出発14日前から市区町村役場に提出可能（マイナポータルでオンライン申請可の自治体あり）
    - 「国外転出届」を提出 → 住民票の除票が実行される
  - **最適な出国タイミング（税務上）**:
    - 12月31日（年末）より前に転出届を提出・除票が完了していること
    - ただし10月〜12月出発が住民税・所得税両面で最もコスト効率が高い
    - 1月に出発すると、前年1月1日時点で住民登録があるため前年分住民税は全額発生
  - **注意点**:
    - 転出後も日本国内で源泉徴収される所得がある場合は「納税管理人」の届出が必要
    - 社会保険（国民健康保険・国民年金）は転出届と同時に脱退手続き
    - 転出後に一時帰国しても1月1日時点で住民票がなければ翌年分住民税は発生しない
    - 帰国が1月1日をまたぐ場合は特に注意（住民票を戻すタイミングで課税年度が変わる）

## 情報10: 日本の「非居住者」認定の実態（183日ルールの誤解）

- 出典: https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/2012.htm / https://probitas.jp/kokusaizeimu/kojinmuke/residentnonresident/ / https://tomurazeirishi.com/where-is-the-condition-address-of-non-residents-in-japan/
- 信頼度: 高（国税庁公式含む）
- 内容:
  - 日本の所得税法は「183日以上海外にいれば非居住者」という規定は**存在しない**
  - 「居住者」の定義: 国内に「住所」（=生活の本拠）を有する、または1年以上居所を有する個人
  - 非居住者認定は「住所がどこにあるか」の実態判断 → **183日は関係ない**
  - **住所の判定要素（総合的に判断）**:
    - 職業・業務の場所
    - 資産の所在地
    - 配偶者・子供の居住地
    - 生活実態（食事・就寝・活動拠点）
    - 日本への帰国頻度・目的
  - 183日ルールは**租税条約**の「短期滞在者免税」に関する条項（例: 雇用所得の源泉国課税免除）→ 居住者判定とは別概念
  - **税務調査リスク**: 住民票を抜いても日本に実態的な生活基盤が残っている場合、「居住者」と認定されるリスクあり
  - **実務上の対策**:
    - 日本の自宅を解約（または完全に第三者に貸し出し）
    - 配偶者・家族も同行または海外拠点に移す
    - 日本への一時帰国を年間183日未満に抑える（実態として有利）
    - UAE税務居住者証明書を取得しておく

    - 日本居住者が100%支配するエストニア法人 → 日本のCFC税制が適用されうる
    - VAT登録後はEU内B2B取引でリバースチャージが複雑化

## 情報12: シンガポール法人の税率・スタートアップ免税・外国人設立要件

- 出典: https://www.jetro.go.jp/world/asia/sg/invest_04.html / https://sinbiz-support.com/sinbizhp/merit-demerit/ / https://www.ey.com/ja_jp/technical/japan-business-services/jbs-insights/accounting-auditing-tax-guides/ey-japan-jbs-singapore-guide-2025
- 信頼度: 高（ジェトロ・EY公式）
- 内容:
  - **法人税率**: 一律**17%**（フラット税率）
  - **スタートアップ税免除制度（SUTE）**:
    - 設立から連続3年間適用
    - 最初のSGD 100,000の課税所得: **75%免税**（実質4.25%）
    - 次のSGD 100,000: **50%免税**（実質8.5%）
    - 上記の恩恵は製造業除く全業種適用可（一定条件あり）
  - **2025年税制改正**: CIT Rebate として法人に SGD 2,000の現金給付 + SGD 40,000上限で50%税額控除あり
  - **外国人設立の要件**:
    - シンガポール居住の取締役が最低1名必要
    - 自前で確保できない場合はノミニーディレクター（名義取締役）サービス利用: 年間SGD 2,400〜6,000
    - 2025年6月以降: ACRA新規定でノミニーディレクター任命に背景情報届出が必要
  - **コスト**: 設立費用SGD 300〜1,000 + ノミニー代行費 + 会計費 → 年間総額SGD 5,000〜15,000（約50〜150万円）
  - **海外所得の取扱い**: テリトリアル課税制度 → シンガポール源泉所得のみ課税。海外子会社からの配当は一定条件で免税
  - 日本のCFC税制: シンガポール法人（税率17%）はトリガー税率20%を下回るため要注意

## 情報13: ジョージア（グルジア）法人設立・税制詳細

- 出典: https://all-about.ge/tax_system/ / https://offshore-houjin.com/taxsavingknowledge/georgiaoffshore/ / https://pineconeblog.com/crypto-georgia-tax/
- 信頼度: 中
- 内容:
  - **法人税**: 一律**15%**（フラット）。2017年以降エストニア型分配課税方式を採用 → **利益留保中は0%**、配当分配時に15%
  - **個人所得税**: 一律**20%**（フラット）

  - **IT Virtual Zone（VZ）制度**:
    - 海外向けソフトウェア開発専業の法人に限定（2021〜2022年以降規制強化）
    - 適格収入に対し法人税・配当税免除。ただし「ソフトウェア開発」のみ対象（マーケティング等は不可）
  - **仮想通貨課税**:
    - 個人のビットコイン等売却益: **非課税**（183日以上のジョージア居住者が対象）
    - 法人からの仮想通貨収益は通常の法人税対象
  - **海外所得の取扱い**: 外国源泉所得は一定条件で非課税（テリトリアル課税的要素）
  - **法人設立コスト**: 最安クラス。設立自体は数百ドル相当。ノマドの拠点国として急成長
  - 日本のCFC税制: ジョージア法人も15%税率でトリガー20%未満 → 日本居住者は要注意
  - **ノマドビザ**: 2026年版でも継続提供中。月収3,000USD以上が条件とされる

## 情報15: BVI・オフショア法人とCFCリスク（日本人向け分析）

- 出典: https://globalwealthprotection.com/top-countries-with-no-cfc-rules-for-digital-nomads/ / https://www.offshore-protection.com/offshore-blog/offshore-company-digital-nomads-perpetual-travelers / https://www.jwing.net
- 信頼度: 中
- 内容:
  - **BVI（英領ヴァージン諸島）**:
    - 法人税率: **0%**
    - 年間維持費: $2,000〜6,000
    - 世界のオフショア法人の約41%がBVIに登録
    - BVI自体にCFC規定なし → 但し日本居住者には日本のCFC税制が適用（意味なし）
  - **BVIが日本人に事実上使えない理由**:
    - 実体なし法人 → 経済活動基準（4条件）を満たせない → 日本のCFC税制で全所得合算
    - 実質的な回避策は「日本の非居住者になること」のみ
  - **日本人がBVIを合法利用できるケース**: 日本非居住者で居住国（UAE等）に税務上の居住がある場合
  - **ジブラルタル**:
    - 法人税率: **10%**
    - 欧州系企業・暗号資産事業者に人気
    - 英国のブレグジット後も英国系法制を維持
    - 日本のCFC税制: 10%税率でトリガー20%未満 → 日本居住者には要注意
  - **ケイマン諸島**: 法人税0%。金融・ファンド向けが主流。設立維持費が高め
  - **総括**: 日本居住者がオフショア法人を設立しても節税にならない。有効活用には非居住者化が必須条件

## 情報16: UAE個人所得税ゼロと日本人の非居住者要件

- 出典: https://bitex-dxb.com/2025/1134/ / https://note.com/g3pdubai/n/n171ada29bc16 / https://socpa.world/blog/nonresident/
- 信頼度: 中〜高
- 内容:
  - **UAE個人所得税**: **0%**（給与・事業所得・投資利益・仮想通貨売却益すべて非課税）
  - **UAE VAT**: 5%（2018年導入、個人消費には関係）
  - **UAE税務居住者の認定基準（2023年制度導入）**:
    - 183日以上滞在、または
    - 90日以上滞在 + 居住許可（ビザ）+ 恒久的住居の所有/賃借
  - **UAE税務居住者証明書（TRC）**:
    - FTA（連邦税務局）に申請可能
    - 日本側に「UAE居住者」であることを証明する書類として活用
    - 特にドバイの銀行口座保有・ビザ保有・183日以上の滞在が条件として有効
  - **日本の非居住者になるための実務チェックリスト**:
    1. 日本の住民票を除票（国外転出届提出）
    2. 日本の賃貸物件を解約（または完全に貸し出し）
    3. 配偶者・子供も移住（または関係が独立しているか整理）
    4. UAE居住ビザ取得（フリーゾーン法人ビザ or ゴールデンビザ）
    5. UAE TRC取得
    6. 日本への帰国は生活の本拠が日本にないことを担保できる範囲で
  - 「住民票を抜くだけ」では不十分。実態的な日本からの生活拠点移転が必要

## 情報17: 令和7年（2025年）税制改正 CFC関連と海外移住への影響

- 出典: https://legacy.ne.jp/knowledge/before/fuyuusou/744-gaikoku-kogaisha-gassan-zeisei-kaisei-saishin-jouhou/ / https://www.meti.go.jp/policy/external_economy/toshi/kokusaisozei/cfc/PDF/170822_cfc.pdf
- 信頼度: 高
- 内容:
  - **令和7年改正CFC関連の主な変更**:
    - 外国関係会社に係る**課税対象金額等の合算時期**: 従来の「事業年度終了後2ヶ月」→「**4ヶ月**」に延長
    - これにより申告準備期間が実務上拡充（対応の余裕が増えた）
  - **個人の海外転出への影響**:
    - 出国税（国外転出時課税）の対象資産の範囲: 2025年時点で暗号資産は対象外維持
    - ただし政府税調で暗号資産を将来的に対象化する議論が継続中
  - **海外移住前の重要確認点**:
    - 出国税対象かどうか（資産1億円以上、国内居住5年超）
    - CFC税制の対象となる外国法人株式を保有していないか
    - 国外転出後も日本で発生する所得（不動産賃料・配当等）に対する源泉税の確認
    - 出国前に税理士（国際税務専門）への相談を強く推奨
