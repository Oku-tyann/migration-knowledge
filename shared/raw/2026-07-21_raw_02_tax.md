## 情報9: タイ 国外源泉所得課税 2024年改正の実態
- 出典: WebSearch（yamada-partners.jp / bangken.com / a-and-aa.com / bdo-japan.jp）
- 信頼度: 高
- 内容:
  - **旧ルール（〜2023年12月31日）**：タイ居住者が国外で得た所得を「所得発生と同じ暦年内」にタイへ持ち込んだ場合のみ課税。翌年以降に送金すれば免税だった
  - **新ルール（2024年1月1日〜）**：タイ居住者が国外源泉所得をタイへ送金・持ち込みした年に、その暦年内に発生した所得として個人所得税課税。発生年と送金年を一致させる「裏技」は封鎖
  - **例外①**：2024年1月1日より前に発生した所得には新ルール不適用（経過措置）
  - **例外②**：タイ非居住者（暦年中180日未満滞在）には適用なし
  - **例外③**：「所得」でないもの（元本、贈与、相続財産等）は対象外
  - **183日ルールについて**：タイの条文は180日。国際的文脈で183日とも言われるが、タイ税法は暦年中「180日以上」滞在を居住者と定義
  - **日本との二重課税**：日タイ租税条約により一定の二重課税救済あり。日本で課税された場合、タイでの外国税額控除適用可能性がある
  - **実務的影響**：2025年以降のタイ在住日本人は、海外の投資・事業所得を毎年の税申告に組み込む必要あり。税理士への相談が必須

## 情報10: タイ 183日ルールの回避・管理方法
- 出典: WebSearch（kyujin.careerlink.asia / a-and-aa.com / zero-asia.biz）
- 信頼度: 中
- 内容:
  - 暦年（1月1日〜12月31日）ベースで滞在日数をカウント。連続した滞在か複数入国かは問わない
  - 180日を超えると「タイ税法上の居住者」となり、国外所得のタイ持ち込み分が課税対象
  - **対策①**：年間滞在日数を179日以内に抑える（別の国をベースにする）
  - **対策②**：タイへの送金を元本・貯蓄から行い、当年発生の「所得」の送金を最小化する
  - **対策③**：タイ国内で稼得した所得とタイ国外所得を明確に分離・記録する
  - **注意**：2024年改正の運用実態・税務調査強化度合いはまだ不透明な部分が多い

## 情報11: 日本の出国税（国外転出時課税）の適用条件
- 出典: 国税庁公式（nta.go.jp）/ WebSearch（ht-tax.or.jp / mac-g.co.jp / tomorrowstax.com）
- 信頼度: 高
- 内容:
  - **制度開始**：2015年7月1日以降の国外転出から適用
  - **適用要件**（両方を満たす場合に課税）：
    1. 国外転出日時点で時価合計1億円以上の「対象資産」を所有
    2. 国外転出日以前10年以内に、日本国内に住所または居所を5年超有していた
  - **対象資産**：有価証券等（株式・投資信託・デリバティブ等）の含み益。不動産は対象外
  - **税率**：所得税15.315%＋住民税5%＝計20.315%（復興特別所得税込）
  - **納税猶予**：納税管理人の届出＋担保提供で最長5年（申請により最長10年）の猶予可能
  - **帰国免税**：国外転出から5年以内（猶予延長した場合は10年以内）に帰国し、かつ対象資産を引き続き保有している場合は課税がなかったものとして取消し
  - **贈与・相続への拡張**：日本居住者が非居住者へ対象資産を贈与・遺贈した場合にも同様の課税あり

## 情報12: 日本の出国税 対策・回避方法
- 出典: WebSearch（probitas.jp / shiodome.co.jp / smbc.co.jp）
- 信頼度: 中〜高
- 内容:
  - **対策①（含み益圧縮）**：出国前に含み益のある資産を売却し課税関係を確定させる（出国後の売却との税負担比較が必要）
  - **対策②（資産分散）**：対象資産の時価合計を1億円未満に抑える（NISA口座内の資産は計算対象外）
  - **対策③（納税猶予の活用）**：担保提供で最長10年猶予。猶予中に対象資産の価値が下落した場合は減額請求可能
  - **対策④（帰国見込みがある場合）**：5年以内帰国で課税取消しの活用を計画に組み込む
  - **対策⑤（早期の海外移住）**：10年以内に日本在住5年超を満たす前に出国する（若い世代向け）
  - **注意**：NISA口座内資産は出国税の計算対象外だが、非居住者になるとNISA口座は維持できない

## 情報13: CFC税制（タックスヘイブン対策税制）のドバイ法人への適用
- 出典: WebSearch（jetro.go.jp / socpa.world / meti.go.jp / agsc.co.jp）
- 信頼度: 高
- 内容:
  - **制度概要**：日本の法人・個人が外国子会社（持分50%超）を保有し、その子会社の実効税率が「トリガー税率（20%）」未満の場合、子会社所得を日本側所得に合算して日本で課税
  - **UAEの法人税率は9%** → トリガー税率20%を大幅に下回るため、ドバイ法人は原則としてCFC税制の対象
  - **個人（非居住者）への適用**：CFCは法人向けが主だが、日本の個人居住者でも特定外国子会社の規定（租税特別措置法40条の4等）で同様の合算課税あり
  - **除外要件（経済活動基準）**：以下4基準すべてを満たせば合算課税を回避可能
    1. **事業基準**：主として株式保有・貸付・特許等の受動的所得を得る事業でない
    2. **実体基準**：本店所在地国に主たる事業に必要な事務所・店舗・工場等を有する
    3. **管理支配基準**：事業の管理・支配・運営を本店所在地国で実際に行う
    4. **所在地国基準**（or 非関連者基準）：主として本店所在地国での事業活動
  - **実務上の対策**：UAE内に実際のオフィス・従業員・取引実態を置く。日本から遠隔で「管理」している状態ではCFC適用リスク大

## 情報14: 日本の住民票を抜く最適タイミング
- 出典: WebSearch（mailmate.jp / taxlawyer328.jp / moneyforward.com / bs.taxlawyer328.jp）
- 信頼度: 高
- 内容:
  - **住民税の「1月1日ルール」**：その年1月1日時点で日本に住民登録がある者に、前年の所得に対する住民税が課税される（翌年6月から納付）
  - **2025年1月1日に住民票がある場合**：2024年分所得の住民税（約10%）が2025年6月〜2026年5月にかけて課税される
  - **最適な住民票除票タイミング**：
    - 翌年の住民税を完全回避したい場合 → **12月31日までに「海外転出届」を提出し住民票を除票**
    - 例：2025年中に海外移住する場合、2025年12月31日までに転出届を出せば2026年度住民税（2025年分所得）は非課税
  - **所得税との違い**：所得税は暦年ベース＋居住者判定で決まる。12月31日時点で非居住者であれば、その年の所得税は非居住者税率（源泉分離課税）
  - **帰国時の注意**：1月1日以降に帰国・転入届を出せば、その年の住民税は課税されない。1月1日当日や前日に転入するのは最悪のタイミング
  - **実務的な手順**：① 海外渡航前に市区町村窓口へ「海外転出届」提出（渡航1〜2週間前から受付可）② 国民健康保険・国民年金は同時に脱退可能 ③ 住民基本台帳ネットワークから除票

## 情報15: 日本住民票除票後の注意点・デメリット
- 出典: WebSearch（nativecamp.net / academia-sch.jp / tax.prime-partners.co.jp）
- 信頼度: 高
- 内容:
  - **メリット**：住民税非課税、国民健康保険料支払い不要、国民年金は任意加入となる
  - **デメリット①**：日本の健康保険（国保・社保）が使えなくなる → 海外旅行保険または現地保険への加入が必要
  - **デメリット②**：マイナンバーカードの機能制限。e-Taxが一部不便になる

  - **デメリット④**：印鑑証明、住民票の写しが取得不可 → 不動産取引・法人登記等に支障
  - **デメリット⑤**：一部の自治体サービス（予防接種等）が利用不可
  - **節税効果の試算**：住民税10%+所得税（累進）= 課税所得1,000万円の場合、住民税約100万円/年が節約可能

## 情報16: 法人税制比較における日本CFC対策のまとめ
- 出典: WebSearch（socpa.world / oshikata-tp.com）
- 信頼度: 高
- 内容:
  - シンガポール（17%）は20%トリガー未満→CFC対象。ただし本物の事業活動があれば経済活動基準で除外可
  - エストニア（20%）は配当課税時にちょうどトリガーに相当。ただし留保利益の実効税率0%問題あり
  - マルタ（名目35%）→ 表面上はトリガー超過でCFC非対象。ただし還付後の実効税率5%は「実質」税率としてCFCの論点あり（税務当局との解釈リスク）
  - ジョージア（バーチャルゾーン0%）→ 明らかにCFC対象。経済活動基準での除外が唯一の対策
  - UAE/ドバイ（9%）→ CFC対象。経済活動基準（UAE内に実体）で除外を目指す
  - **結論**：どの国でも「実体のある事業」がないと日本CFC税制を回避できない。法人税率だけを見た節税は2025年以降ほぼ不可能
