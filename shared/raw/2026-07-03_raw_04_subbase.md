## 東南アジア系

### 1. マレーシア（クアラルンプール）

**生活費（月額目安）**
- 快適な生活: RM 6,000〜10,000（約20〜33万円）
- コンドミニアム2LDK（プール・ジム付き）: RM 3,500〜5,000（約12〜17万円）
- 日本食材・和食レストランは日本の1.2〜1.5倍
- 移住初期費用（ビザ取得・引越・家具等）: 最低400万円前後

**治安**
- 東南アジアの中では比較的治安良好
- 強盗発生率は日本の約25倍
- スリ・置き引きに要注意
- セキュリティ付きコンドミニアム居住を推奨

**ビザ**
- MM2H（Malaysia My Second Home）: 長期居住ビザ
- デジタルノマドビザあり
- 注意点: 2022年以降MM2Hの条件が厳格化

**気候**
- 熱帯性気候、年間を通じて高温多湿（28〜35℃）
- 雨季・乾季あり（東海岸と西海岸で異なる）

**インターネット環境**
- 光ファイバー普及、都市部は高速
- TM Unifi等が主要プロバイダ

**日本人コミュニティ**
- KL在住日本人数は約2万人超
- 日本人学校・日本食レストラン・日本語スーパー充実

**総合評価**: ⭐⭐⭐⭐☆ — 東南アジアで最もコスパが高い移住先のひとつ。英語通用度高く生活インフラ整備済み。ただし物価は上昇傾向。

### 2. ベトナム（ホーチミンシティ）

**生活費（月額目安）**
- ノマド最低限: 約15〜20万円
- 家賃（市内1LDK）: $500〜900
- 食費: 現地食中心なら月2〜3万円程度
- ダナンなら家賃3〜4万円、食費月2万円台も可能

**治安**
- 親日国で比較的安全
- スリ・バイクひったくりに注意（観光地中心）
- ホーチミンシティは交通量が多く事故リスクあり

**ビザ**
- 日本国籍: 45日間ビザなし入国可（2023年改正）
- 長期滞在: 就労ビザ（Eビザ）＋労働許可証が主流
- 2025年4月: 「10年ゴールデンビザ」導入提案（未確定）
- デジタルノマド専用ビザは現時点では未整備
- 永住権取得は要件厳しく困難

**気候**
- 熱帯モンスーン気候
- ホーチミンは雨季（5〜10月）と乾季（11〜4月）
- 年間平均気温27〜30℃

**インターネット環境**
- 都市部は高速ファイバー普及
- コワーキングスペースも充実

**日本人コミュニティ**
- ホーチミン・ハノイとも日本人多数
- 日系企業多数進出（就職ルートあり）
- 日本語教育への需要高く、親日感情強い

**総合評価**: ⭐⭐⭐☆☆ — 生活費は安くコスパ良好だが、長期滞在向けビザ整備が課題。ゴールデンビザ実現次第で評価上昇の可能性大。

### 3. フィリピン（セブ）

**生活費（月額目安）**
- 快適生活: ₱60,000〜90,000（$1,000〜1,600）
- 住宅（IT Park・Lahug・Business Park 1BR）: $300〜600
- 郊外1BR: $200〜400
- 電気代: ₱5,000〜8,000/月（高め）
- ネット（光ファイバー）: ₱1,200〜2,000/月

**治安**
- セブ市内の観光・ビジネス地区は比較的安全
- 地域格差大きい（貧困地区・夜間の単独行動は注意）
- 欧米・日本人向けエリア（IT Park周辺）は管理良好

**ビザ**
- 2025年6月: デジタルノマドビザ（DNV）正式開始（大統領令86号）
- フィリピンは東南アジアで数少ないDNV保有国
- リタイアメントビザ（SRRV）: 35歳以上、$10,000〜20,000預金が条件

**気候**
- 熱帯性気候、年間高温
- 台風シーズン（6〜11月）に注意
- セブは直接台風を受けにくい地理的位置

**インターネット環境**
- IT Parkエリアは最大1Gbpsの光ファイバー
- PLDT・Globe・Smart・Converge ICT等
- コワーキングスペースが充実

**日本人コミュニティ**
- 日本人留学生・在住者は増加傾向
- 語学留学で日本人が多いため日本語サービス充実
- 日系企業はマニラ中心だがセブでも増加

**総合評価**: ⭐⭐⭐⭐☆ — 英語環境・ノマドビザ・温暖な気候が魅力。台風リスクと停電頻度がマイナス点。

### 4. インドネシア（バリ）

**生活費（月額目安）**
- ミニマム生活: $900〜1,200（約14〜19万円）
- 快適生活（Canggu・Ubud等 1BR villa）: $1,800〜2,500
- 豊かな生活: $2,500〜3,500
- 西洋の同水準生活比: 50〜65%安

**内訳（快適生活想定）**
- 住宅（1BR villa, AC/プール付き）: $700〜1,400
- 食費（外食中心）: $300〜500
- 交通（スクーター+ガソリン）: $50〜100

**治安**
- バリ自体は比較的安全な観光地
- 観光客向けの詐欺・ぼったくりに注意
- 麻薬密輸に対しては死刑を含む厳しい処罰

**ビザ**
- B211A観光ビザ: 60日（30日×2延長可、最大60日延長）
- デジタルノマドビザ（E33G）: 5年間、海外収入のみなら税免除
- ゴールデンビザ: $350,000以上の投資が必要
- 長期滞在にはKITAS（一時滞在許可）が必要

**気候**
- 熱帯性気候
- 乾季（4〜10月）・雨季（11〜3月）
- 年間平均26〜32℃

**インターネット環境**
- Canggu・Ubud等ノマド集積地ではカフェ・コワーキング充実
- 光ファイバーは普及中だが農村部は弱い

**日本人コミュニティ**
- バリ在住日本人は近年増加、特にCanggu・Seminyak
- 日本語対応の不動産会社・医療機関あり

**総合評価**: ⭐⭐⭐⭐☆ — ノマド聖地としての地位確立。ただし観光ビザ連続取得の難易度上昇、インフラ整備は都市部のみ。

### 5. カンボジア（プノンペン）

**生活費（月額目安）**
- 節約生活: 月5〜8万円も可能
- 快適生活（2人暮らし）: $1,000〜1,200（約15〜18万円）
- コンドミニアム（清潔・1BR）: $400〜500/月
- 物価は日本の約1/3

**治安**
- 普通の生活では大きな問題なし
- 2023年: プノンペンで日本人犯罪被害8件（大使館統計）
- スリ・バイクひったくりに注意
- 新国際空港「テチョ・クロン・タクマウ」が2025年9月開港予定（アクセス改善）

**ビザ**
- Eビザ（業務ビザ）: 収入・学歴審査なし、年次更新で長期滞在可能
- リタイアメントビザ（ERビザ）: 55歳以上、約$300
- 永住権: 一定条件で取得可能（ER-PRビザ等）
- 周辺国最易クラスのビザ取得しやすさ

**気候**
- 熱帯モンスーン気候
- 乾季（11〜4月）・雨季（5〜10月）
- 年間平均27〜35℃

**インターネット環境**
- プノンペン市内は光ファイバー整備中
- コワーキングスペースは増加傾向だが少ない

**日本人コミュニティ**
- 日本人在住者は数千人規模
- 日系企業・NGO進出あり、コミュニティ形成済み

**総合評価**: ⭐⭐⭐☆☆ — 生活費最安クラスでビザ取得も容易。一方で医療インフラ・生活利便性はまだ発展途上。

## 地中海系

### 6. ポルトガル（リスボン）

**生活費（月額目安）**
- 全体: 20〜30万円（€1,000〜1,500）
- 住宅（市内1BR）: €800〜1,500、郊外: €500〜700
- 食費: €200〜300/月
- 光熱費・通信費: €100未満/月
- 民間保険: 年€300〜1,000

**治安**
- 西欧の中では治安良好
- 観光地でのスリ・置き引きに注意
- 全体的に安全で住みやすい

**ビザ**
- デジタルノマドビザ（D8ビザ）: 最低収入€3,480/月（2025年基準）
- Dビザ/パッシブインカムビザ: リタイア・不労所得者向け
- ゴールデンビザ: 不動産投資500,000ユーロ等
- NHR税制（非常居住者制度）: 廃止→後継制度「IFICI」に移行

**気候**
- 地中海性気候
- 夏（6〜9月）: 25〜35℃、晴天多い
- 冬（12〜2月）: 10〜15℃、温暖
- 年間日照時間約3,000時間

**インターネット環境**
- 光ファイバー普及率高い（EU内トップクラス）
- コワーキングスペース増加中

**日本人コミュニティ**
- リスボン在住日本人数は少数だが増加傾向
- SNS中心の情報共有コミュニティあり
- 日本食レストラン数は限定的

**総合評価**: ⭐⭐⭐⭐☆ — 欧州移住で最もコスパ良好な選択肢の一つ。物価上昇傾向に注意が必要。

### 7. マルタ

**生活費（月額目安）**
- 単身者: €1,700〜2,300/月
- 家賃除く生活費: €760以上/月
- 住宅（市内1BR）: €1,000〜1,500（Sliema・St Julian's等プレミアエリア: €1,100〜1,600）
- Gozo島: €500〜900

**治安**
- EU加盟国で治安良好
- 観光地での軽微な犯罪に注意
- 全体的に安全な環境

**ビザ**
- デジタルノマドビザ（Nomad Residence Permit）: 2021年導入
  - 最低収入: 年€42,000以上（約€3,500/月）
  - 申請費用: €300/申請者、従属家族€300/人、IDカード€100/人
  - 処理期間: 約30営業日
  - 更新条件: 年間最低5ヶ月マルタ滞在
- 外国人人口: 全体の約31%（2025年初頭）

**気候**
- 地中海性気候
- 夏: 30〜35℃
- 冬: 10〜15℃
- 年間日照時間約3,000時間（欧州最高水準）

**インターネット環境**
- 光ファイバー普及
- Valletta・Sliema・St Julian'sにコワーキングスペース集積

**日本人コミュニティ**
- 日本人コミュニティは小規模
- 外国人コミュニティ全体は非常に大きい（外国人31%）
- 英語公用語のため日常生活は英語で可能

**総合評価**: ⭐⭐⭐☆☆ — 欧州内で英語が公用語という強みあり。物価は比較的高め、日本人コミュニティは小さい。

### 8. スペイン（バルセロナ/バレンシア）

**生活費（月額目安）**
- 単身者（家賃除く）: 約€845（約12万円）
- マドリード・バルセロナ 家族向け1,000〜1,800ユーロ（133,000〜240,000円）
- ロンドン・パリより30%程度安い
- バレンシアはバルセロナより15〜25%安い

**治安**
- 全体的に安全だが観光地のスリに注意
- バルセロナのランブラス通り周辺は特にスリが多い
- バレンシアは比較的落ち着いた環境

**ビザ**
- デジタルノマドビザ（2023年導入）
  - 初回: 最長1年（スペイン内での申請可）
  - 更新: 最長5年まで
  - 5年居住後: 永住権申請可
  - 最低収入: EU最低賃金の200%以上（約€2,646/月、2025年基準）
- 非営利活動ビザ・パッシブインカムビザも選択肢

**気候**
- 地中海性気候（バレンシア・バルセロナ共）
- 夏: 25〜35℃
- 冬: 10〜15℃
- バレンシアは年間平均気温が高く日照豊富

**インターネット環境**
- EU内で高水準
- バルセロナは特にテック系スタートアップ集積地
- コワーキングスペース充実

**日本人コミュニティ**
- バルセロナ在住日本人は数千人規模
- 日本語コミュニティ・日本食レストランあり
- バレンシアは日本人コミュニティ小さめ

**総合評価**: ⭐⭐⭐⭐☆ — 欧州生活の質・文化・気候が高バランス。ノマドビザも整備。物価はポルトガルよりやや高め。

### 9. クロアチア

**生活費（月額目安）**
- 農村・内陸部: €1,000未満
- 都市・沿岸部（Split・Dubrovnik等）: €1,800以上
- 首都Zagreb 1BR家賃: €600〜800/月

**治安**
- EU加盟国で治安良好
- 観光地（Dubrovnik等）は夏季混雑
- 全体的に安全

**ビザ**
- デジタルノマドビザ（2021年導入、2025年3月改正）
  - 滞在期間: 18ヶ月（改正前12ヶ月から延長）
  - 最低収入: €3,622.50/月（クロアチア平均賃金の2.5倍）
  - 配偶者同伴: +10%（€3,984.75/月）
  - クロアチア国内所得税: 免除
  - 申請書類: 6ヶ月分の銀行明細または給与明細
  - 改正日: 2025年3月15日施行

**気候**
- 地中海性気候（沿岸部）・大陸性気候（内陸部）
- 夏: 25〜35℃（アドリア海沿岸）
- 冬: 0〜10℃

**インターネット環境**
- EU加盟後インフラ整備進む
- 都市部は高速ブロードバンド普及
- 農村部は整備途上

**日本人コミュニティ**
- 日本人コミュニティは非常に小規模
- 英語は観光業では通じるが、クロアチア語が必要な場面も

**総合評価**: ⭐⭐⭐☆☆ — 自然環境・EU圏・税免除が魅力。最低収入ハードルが高く、日本人コミュニティはほぼなし。

### 10. ジョージア（トビリシ）

**生活費（月額目安）**
- デジタルノマド一般: $1,200〜1,500/月
- 全体平均: $1,569（2026年5月時点）
- 家賃平均: $840/月
  - Saburtalo地区スタジオ: $520/月
  - Vera地区中心部: $890/月
- 食費（外食）: ₺15〜25 GEL（$5〜9）
- 食料品: $120〜200/月

**注意**: 2022年のロシア人大量移住により物価が20〜40%上昇、一部地区では家賃が倍増

**治安**
- 全体的に治安良好
- 夜間のナイトライフエリアは注意
- カフカス地方特有の政情不安に注意（2024年以降の国内政治混乱）

- 日本国籍: 最大1年（365日）ビザなし滞在可
- フリーランサー向け優遇税制: 収入の1%
- 法人設立が容易で外国人でも開設可能
- 永住権取得は条件あり

- 大陸性気候
- 夏: 25〜35℃（乾燥）
- 冬: -5〜5℃（積雪あり）
- 4季がある

**インターネット環境**
- 光ファイバー普及、速度は良好

**日本人コミュニティ**
- 日本人在住者は少ないが増加傾向
- ノマドコミュニティ全体は充実（英語対応可）

**総合評価**: ⭐⭐⭐⭐☆ — 365日ノービザ×1%税率は世界最強クラスのノマド優遇。ただし政治状況と物価上昇には注意。

### 11. ニュージーランド

**生活費（月額目安）**
- 全体（NZD）: NZ$3,500〜5,000/月（約35〜50万円）
- オークランド1BR家賃: NZ$2,000〜2,800/月
- 食費: NZ$400〜800/月
- 生活費は日本と同等〜やや高め

**治安**
- 全体的に安全（世界平和指数上位）
- 自然災害（地震・火山）リスクあり
- 都市部での軽犯罪は一定数あり

**ビザ**
- ワーキングホリデービザ（18〜30歳）: 12ヶ月滞在・就労可
- 観光ビザ（2025年1月27日以降）: リモートワーク可（最大90日、条件次第で9ヶ月まで延長可能）
- デジタルノマド専用ビザ: 現時点で専用制度なし
- 永住権: ポイント制（スキルドマイグラントカテゴリー等）

**気候**
- 南半球のため季節が逆
- 北島（オークランド）: 温暖（10〜25℃）
- 南島（クライストチャーチ）: やや寒冷
- 天候変化が激しい

**インターネット環境**
- 光ファイバー普及
- 速度・安定性良好

**日本人コミュニティ**
- オークランドに日本人コミュニティあり
- ワーホリ日本人多数
- 日本語サービス・日本食レストランあり

**総合評価**: ⭐⭐⭐☆☆ — 生活環境・自然は世界最高水準。ただし生活費高め、専用ノマドビザなし、日本から遠い点がデメリット。

### 12. メキシコ（メキシコシティ）

**生活費（月額目安）**
- 単身（2BR賃貸）: 5,000〜13,000ペソ（約4〜10万円）
- ルームシェア相場: 3,000〜8,000ペソ
- 屋台タコス: 10〜20ペソ×3〜6個 = 60〜120ペソ/食
- 全体的にアジア系より割高傾向の地区もある

**治安**
- 地域格差が非常に大きい
- 麻薬組織抗争が続く州・地区あり
- 2025年総犯罪件数: 約202万件（過去最高水準）
- 日系企業集積エリア・観光地は比較的安定
- 外務省危険情報の確認必須

**ビザ**
- 日本国籍: 180日ビザなし入国可
- テンポラリー・レジデント（一時居住）: 収入証明あれば比較的容易
- パーマネント・レジデント: 4年以上の居住後に申請可
- デジタルノマドビザ: 専用制度なし（観光ビザ内でのリモートワークが事実上黙認）

**気候**
- メキシコシティ（標高2,240m）: 年間15〜25℃と過ごしやすい
- 雨季（5〜10月）・乾季（11〜4月）
- 高地のため紫外線強い

**インターネット環境**
- 都市部は光ファイバー普及
- コワーキングスペース増加中（Roma・Condesa地区中心）

**日本人コミュニティ**
- メキシコ全体で約1万人の日本人在住
- 日本人学校・日本食スーパー・日本語サービスあり
- 1,000社以上の日系企業進出

**総合評価**: ⭐⭐☆☆☆ — 日本人コミュニティと都市インフラは充実。ただし治安の懸念と専用ノマドビザ不在が大きなデメリット。
