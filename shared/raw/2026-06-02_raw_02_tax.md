- 信頼度: 高
- 内容:

## 情報6: マルタ法人 — 6/7払い戻し制度と実効税率（EU域内）
- 出典: https://www.oobac.com/malta-company-jp / https://metiluxpartners.com/vol-1676/ / https://www.eurovisa.jp/マルタ法人設立-事業進出/
- 信頼度: 中（一部コンサル系サイト）
- 内容:
  - **名目法人税率**: **35%**（EU内最高水準）
  - **6/7払い戻し制度**：法人が35%税を支払った後、株主（外国人可）が税務当局に還付申請。納税額の **6/7（約86%）が現金で還付**される
  - 実効税率 = 35% × (1 - 6/7) = **約5%**
  - 還付はキャッシュで銀行振込。通常3〜6ヶ月で実行
  - **100%外国人オーナーシップ可**
  - EU加盟国のため、EU内の顧客・取引先との信頼性・二重課税条約網が充実
  - 設立コスト：約EUR 1,500〜3,000（登記、コンプライアンス費含む）。年間維持費 EUR 3,000〜5,000（会計、登記住所、会社秘書など）
  - **注意点**：実際の設立・運営にはマルタの会計士・法人サービスが必須。還付手続きに専門家費用がかかるため、実効税率5%でも手間とコストが存在する。日本のCFC規制への注意も必要

## 情報8: タイ 2024年 海外所得課税改正 — 183日ルールの実態
- 出典: https://www.yamada-partners.jp/global-report/th-report-240305 / https://bangken.com/column/2024/0305-3041289/ / https://www.ht-tax.or.jp/topics/thai2024-mochikomikazei-rules/ / https://a-and-aa.com/news/2024/01/112/
- 信頼度: 高（大手税理士法人・会計事務所の一次分析）
- 内容:
  - **改正の根拠**：2023年9月、タイ歳入局命令 **Por.161/2566** により、国外源泉所得の持ち込み課税ルールを厳格化
  - **旧ルール**（〜2023年末）：国外所得を「稼得した翌年以降」にタイへ持ち込めば非課税。これを利用したクロスイヤー節税が一般的だった
  - **新ルール**（2024年1月1日〜）：**2024年1月1日以降に稼得した国外所得**は、持ち込み年に関わらず、タイへ送金・持ち込んだ年の個人所得税（PIT）の対象となる
  - **経過措置**：2024年1月1日より前に稼得した所得はタイに持ち込んでも申告不要（2024年1月FAQ）
  - **居住者の定義**：暦年180日（≒183日と通称）以上のタイ滞在者
  - **非居住者**（180日未満）は今回の改正の影響なし。タイ国内源泉所得のみ課税
  - **実務上の課題**：タイには外国税額控除制度があるため、日本で課税済みの所得は二重課税を一定程度回避できるが、タイ税率（最高35%）との差額が生じる可能性
  - **二重課税条約**：日タイ租税条約あり。ただし適用には証明書類が必要

## 情報9: タイ LTRビザ — 海外所得の税制優遇
- 出典: https://asean-jp.info/thailand/thailand-ltr-visa-definitive-guide-2026/ / https://a-and-aa.com/service/ltr/ / https://ltr.in.th/
- 信頼度: 高
- 内容:
  - **LTR（Long-Term Resident）ビザ**：タイ政府が高付加価値外国人向けに設けた10年長期滞在ビザ
  - **4カテゴリー**：
    1. **Wealthy Global Citizens**（資産家）：資産 USD 100万以上 + 過去2年の年間所得 USD 8万以上 + タイへの投資 USD 50万以上
    2. **Wealthy Pensioners**（退職者）：50歳以上、年金等月収 USD 8万以上
    3. **Work-from-Thailand Professionals**（リモートワーカー）：過去2年の勤務先法人が設立5年以上・収益 USD 1.5億以上かつ個人年収 USD 8万以上
    4. **Highly Skilled Professionals**（高度専門家）
  - **税制優遇**：
    - **Wealthy Global Citizens**カテゴリー：海外源泉所得のタイ持ち込みに対して **課税免除**（2024年改正の影響を受けない）
    - **Work-from-Thailand Professionals**カテゴリー：タイ国内就労所得に対し **優遇税率17%**（通常の累進税率 5〜35% の代わり）
  - LTRビザの申請費：THB 50,000（≈ USD 1,400）
  - **注意**：Wealthy Global Citizensの資産・所得要件のハードルが高く、大多数のノマドには該当しない

## 情報10: 日本 出国税（国外転出時課税） — 条件・しきい値
- 出典: https://www.nta.go.jp/taxes/shiraberu/shinkoku/kokugai/01.htm / https://mac-g.co.jp/international-tax/column/102/ / https://japan-dev.com/blog/exit-tax-japan / https://toma.co.jp/en/blog/jtg/basics-exittax-japan/
- 信頼度: 高（国税庁一次情報）
- 内容:
  - **根拠法令**：所得税法第60条の2〜60条の5（平成27年7月1日施行）
  - **適用条件**（2つとも満たす場合）：
    1. 国外転出時点で対象資産の合計額が **1億円以上**
    2. 転出前 **過去10年以内に5年超** 日本の居住者であった
  - **対象資産**：有価証券（株式・投資信託・債券等）、匿名組合の出資持分、未決済の信用取引・デリバティブ取引。**不動産・暗号資産は現時点で対象外**
  - **課税内容**：転出時点の含み益に対して **所得税（15.315%）及び住民税（5%）** が課税。実質約20%
  - **納税猶予制度**：担保提供を条件として最長 **5年間** 猶予可能。帰国予定がある場合はさらに5年延長（計10年）可能
  - **帰国した場合の取消**：5年（または10年）以内に帰国し、かつ対象資産を保有継続していれば課税取消の申請が可能
  - **対策**：①対象資産が1億円未満になるよう事前に売却（特定口座内で損益通算）、②NISAで保有する株式は対象外のため節税効果、③10年以内5年以下の短期居住者は対象外（長期外国人赴任者向け規定）

## 情報11: 日本 CFC税制（タックスヘイブン対策税制） — ドバイ法人への適用条件
- 出典: https://hls-global.ae/cfc-compliance-uae-japan-guide/ / https://hls-global.jp/en/2025/10/10/uae-economic-substance-the-key-to-controlled-foreign-corporation-cfc-compliances-in-japan/ / https://carnet-tax.com/2025/12/10/japans-cfc-tax-haven-rules-for-corporations/ / https://kpmg.com/us/en/taxnewsflash/news/2025/11/japan-guidance-amendments-iir-cfc-regime.html
- 信頼度: 高
- 内容:
  - **根拠法令**：租税特別措置法第66条の6（法人）、第40条の4（個人）
  - **適用対象となる「外国関係会社」の定義**：日本の居住者・内国法人が直接または間接に株式の **50%超** を保有する外国法人。実質支配関係がある場合も含む
  - **租税負担割合の閾値**：
    - ペーパーカンパニー・キャッシュボックス会社の場合：**27%以下**（2024年4月1日以降の日本親会社の事業年度から。旧：30%）
    - 一般の外国子会社の場合：**20%以下** で合算課税の検討対象
  - UAEの法人税（9%）は明らかに27%を下回るため、ドバイフリーゾーン法人は **CFC規制の対象に入る可能性が高い**
  - **適用回避の条件（経済活動基準、4要件すべて満たす必要）**：
    1. **事業基準**：主たる事業が適格な事業活動であること（持株・特許・金融等は除外）
    2. **実体基準**：現地に固定施設（オフィス、設備）があること
    3. **管理支配基準**：事業の管理・支配が現地で行われること（代表者や意思決定が現地に存在）
    4. **所在地国基準 or 非関連者基準**：主要な取引相手が関連者でないこと等
  - **実務上の対策**：ドバイに実質的なオフィス、現地スタッフ（または自身の滞在）、意思決定プロセスの現地化が必要。「名義だけドバイ法人で自分は日本居住」はCFC適用リスクが極めて高い
  - **個人への適用**：日本居住の個人株主も（法人株主同様）外国子会社の利益が合算される

## 情報12: 日本 住民票を抜くタイミング — 住民税の1月1日ルール
- 出典: https://bs.taxlawyer328.jp/blog/moving-abroad/p12862/ / https://biz.moneyforward.com/establish/basic/81699/ / https://mailmate.jp/ja/blog/kaigaitenshutsu-juminzei
- 信頼度: 高
- 内容:
  - **根拠法令**：地方税法第294条（住民税の賦課期日は1月1日）
  - **1月1日ルール**：住民税は **1月1日時点** に日本に住所を有していた場合にその年度（4月〜翌3月）分が課税される
  - **最適な住民票除票のタイミング**：**12月31日以前に出国・転出届を提出**すれば、翌年度（翌年4月〜）の住民税は発生しない
  - **1月1日以降に転出した場合**：その年の1月1日時点で居住者とみなされるため、その年度分の住民税が1年分課税される
  - **手続き方法**：転出予定日の14日前〜出国当日までに市区町村役場で「国外転出届（海外転出届）」を提出
  - **住民票を抜くメリット**：住民税・国民健康保険料の支払い義務消滅
  - **デメリット**：①国民健康保険の脱退（一時帰国時は民間保険が必要）、②国民年金は任意加入となる（将来の受取額が減少）、③日本の公共サービス（マイナンバーカードの利用制限等）
  - **注意**：住民票を抜いても所得税上の「居住者」判定は別途行われる。住民票除票後も実態として日本に居住している場合、国税当局は独自基準（生活の本拠地）で居住者認定する可能性がある
  - **節税効果の試算例**：大きな売却益（株式・不動産）が発生する年の12月中に転出できれば、住民税5%相当（数十万〜数百万円）を回避できる

## 情報13: 日本 所得税の居住者判定 — 「生活の本拠地」基準
- 出典: https://www.ht-tax.or.jp/sozoku-guide/overseas-migration-tax / https://www.taxlawyer328.jp/inheritance-columns/海外移住/p7192/
- 信頼度: 高
- 内容:
  - 日本の所得税における居住者は「**国内に住所を有し、または現在まで引き続いて1年以上居所を有する個人**」（所得税法第2条）
  - 住所とは「生活の本拠」であり、住民票の有無は一つの要素に過ぎない
  - **判断基準**（国税庁のチェックリスト）：家族（配偶者・子）が日本にいるか、日本に資産（不動産・預金口座）があるか、日本への帰国頻度、職業・契約の所在地など
  - **非居住者とみなされる目安**：①1年以上の海外在住、②海外での職業・ビジネスが主体、③家族も同行、④日本の住居を手放す
  - **住民票除票後も居住者とみなされるリスク**：配偶者が日本に残留、日本の不動産を保有、頻繁な一時帰国などがある場合
  - 国税当局が「非居住者性」を否認した判例も存在。慎重な準備が必要

## 情報16: UAE 2025年 法人税 — フリーゾーン適格所得の最新基準
- 出典: https://tax.gov.ae/en/media.centre/news/federal.tax.authority.issues.corporate.tax.guide.on.free.zone.persons.aspx / https://www.spcfz.ae/blogs/corporate-tax-free-zone-uae/
- 信頼度: 高（UAEの連邦税務局一次情報）
- 内容:
  - 2023年6月施行のUAE Corporate Tax Law（Federal Decree-Law No. 47 of 2022）に基づく
  - **QFZP（Qualifying Free Zone Person）の要件**（すべて満たす必要）：
    1. フリーゾーンまたはその外（海外）で事業を行っている
    2. 十分な実体（Adequate Substance）をフリーゾーン内に有する
    3. 適格所得を得ている
    4. 非適格な収益が全収益の5%以下（De minimis rule）
    5. 規定の財務諸表を作成・提出している
  - **適格収益**の具体例：フリーゾーン外の外国企業との取引収益、フリーゾーン内企業間取引、特定の知的財産所得
  - **9%が課される非適格収益**の例：UAEメインランドの個人・法人との取引、一部の金融・不動産取引
  - **De Minimis規則**：非適格収益が総収益の5%以下かつ AED 5,000,000以下であれば、全収益を適格所得として0%処理可能
  - 2025年の変更：FTAは新たなガイダンスを発出し、「実体」要件として従業員数・オフィス規模・支出水準の具体的なベンチマークを明示
