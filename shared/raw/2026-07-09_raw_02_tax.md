# 税制・法人調査 — 2026-07-09

## 情報1: ドバイ法人設立の概要と税制メリット
- 出典: https://nexpert-law.com/web3/dubai-company-establishment-immigration
- 信頼度: 高
- 内容: ドバイでは法人税が9%（UAE法人税2023年導入）で、日本の法人税23.2%と比較して大幅に低い。個人所得税・住民税なし、付加価値税5%。フリーゾーン法人はリモート電子申請で設立可能だが、銀行口座・エミレーツID取得のため設立後に渡航が必要なケースが多い。2020年12月以降、メインランド法人も外資100%保有が可能。

## 情報2: ドバイ フリーゾーン vs メインランドの選択
- 出典: https://getfr33.com/company-incorporation-fees-dubai/
- 信頼度: 高
- 内容: フリーゾーン法人はUAE国内取引に制限あり（直接取引不可、代理店経由が必要）。メインランド法人はUAE国内外どちらとも取引可能。ノマド・オンラインビジネスで日本人が選ぶ場合、フリーゾーンが主流。IFZA・RAKEZ・SHAMSが低コストで人気。

- 信頼度: 高
- 内容: フリーゾーン法人の基本設立コストはAED 12,500〜50,000（約51万〜206万円）。最安はMeydan Free Zone（AED 12,500）およびIFZA（AED 12,900≒約53万円）。ビザ費用はAED 3,800〜6,500（エミレーツID・健康診断・保険含む）。フレキシデスク（バーチャルオフィス）年間AED 5,000〜15,000。標準的な登録所要期間は書類承認後1〜3営業日。

## 情報5: シンガポール法人税と優遇制度
- 出典: https://www.jetro.go.jp/world/asia/sg/invest_04.html / https://singalife-biz.com/guide/tax2023-cpa/
- 信頼度: 高
- 内容: シンガポール法人税率は一律17%。ただし新設会社向け部分免税制度あり——最初の10万SGDの75%、次の10万SGDの50%が非課税。実効税率はさらに低下。個人所得税最高24%（2023年から）。GST（消費税）は2024年より9%に引き上げ。居住法人のみ各種優遇措置（国外源泉所得免税・租税条約など）を享受可能。

## 情報6: シンガポール法人設立 日本人向けコストと条件

- 信頼度: 高
- 内容: 最低資本金はSGD 1（約110円）と非常に低い。設立自体に約1週間。ただしシンガポールで働く場合はEntrePassまたはEmployment Passが必要。EntrePassは資本金SGD 50,000以上の事業計画審査あり、申請代行費SGD 2,500〜（約27.5万円〜）。現地取締役（シンガポール市民/PR/EP保有者）の1名設置が法的必須。全体所要期間は2〜3ヶ月が現実的。

## 情報7: エストニア e-Residency 法人設立コストと実務
- 出典: https://atsoho.com/blog/estonia-e-residency-merit / https://wise.com/jp/blog/starting-a-business-in-estonia-guide
- 信頼度: 高
- 内容: e-Residency申請費：100〜120ユーロ（約1.6万円）。受け取りは東京の駐日エストニア大使館。法人登録費用：265ユーロ。コンタクトパーソン（エストニア在住者）との年間契約費：200〜400ユーロ。会計代行（Xoloなど）：月額約100ユーロ〜。年間維持費総計：約500〜1,200ユーロ（約8〜19万円）。日本にいながら完全オンラインでEU法人が設立可能。法人税は分配時のみ課税（エストニア型留保利益非課税）。

## 情報8: エストニア法人の税制特性
- 出典: https://www.oobac.com/estonia-company-jp / https://atsoho.com/blog/estonia-e-residency-merit
- 信頼度: 高
- 内容: エストニアの法人税は「配当課税型」——利益を内部留保している限り課税なし。配当分配時に20%課税。EU加盟国のためEU域内での取引・銀行口座開設（Wise Business等）が容易。ただしUAE・ドバイとは異なり、個人がエストニア非居住のままでは個人所得税の節税効果は限定的。

## 情報9: ジョージア法人設立と税制優遇
- 出典: https://atsoho.com/blog/georgia-hojin-setsuritsu-setsuzei / https://all-about.ge/tax_system/
- 信頼度: 高
- 内容: ジョージアの通常法人税は15%（エストニア型——留保利益は非課税、分配時のみ課税）。スモールビジネスステータス取得で年商500,000ラリ（約2,700万円）以下の売上に対し1%の超低税率。ただしスモールビジネスステータスは年間183日以上のジョージア滞在が必要条件。IT Virtual Zone法人は法人税0%（ジョージア外への売上のみ適用）。法人設立費はドバイより大幅に安く、ほぼ無料〜数万円レベルとされる。

## 情報10: マルタ法人設立と実効税率5%の仕組み
- 出典: https://metiluxpartners.com/vol-1676 / https://www.la-quarta.jp/post/malta-tax-system
- 信頼度: 中
- 内容: マルタの法人税は名目35%だが、外国株主への配当時に6/7の還付制度（Tax Refund System）があるため実効税率は約5%まで低下。EU加盟国でありながら実質的な低税率を実現。有限会社設立の最低資本金は1,135ユーロ。2025年3月より全法人設立書類のオンライン提出必須化・24時間設立が可能。70ヶ国以上と租税条約締結済み。ノマドレジデンス許可制度（2024年改定）あり。

## 情報11: BVI（英領バージン諸島）オフショア法人の現状2025
- 出典: https://nomadcapitalist.com/finance/offshore/best-countries-for-offshore-companies/ / https://www.expanship.com/vg/blog/british-virgin-islands-tax-guide
- 信頼度: 中
- 内容: BVI法人は所得税・法人税・キャピタルゲイン税・配当税・源泉徴収税すべて0%（現地税務）。アクティブ法人数は2025年9月時点で361,747社（住人口約32,000人に対し約11倍）。2025年6月にFATFグレーリスト入り（実質的所有者の透明性問題）。OECDピラー2（2024年施行・最低税率15%）により、売上7億5,000万ユーロ超のグループ企業は親会社管轄国でトップアップ課税対象。小規模ノマドへの直接影響は現時点では限定的だが、信用度・銀行口座開設への影響に注意。

## 情報12: タイ 海外所得課税 2024年改正の実態
- 出典: https://www.yamada-partners.jp/global-report/th-report-240305 / https://www.ht-tax.or.jp/topics/thai2024-mochikomikazei-rules/
- 信頼度: 高
- 内容: 2023年9月、タイ歳入局が国外源泉所得課税の通達を発布。2024年1月1日より施行。改正前は「翌年に持ち込めばタイで非課税」という抜け穴があったが、改正後は「2024年1月1日以降に発生した国外所得をタイに持ち込んだ場合（同年内・翌年いずれも）は個人所得税課税対象」。183日ルールは従来通り（タイ居住者＝年間183日以上滞在）。経過措置：2024年1月1日より前に発生した所得には新ルール不適用。

## 情報13: タイ 183日ルールの課税詳細
- 出典: https://a-and-aa.com/news/2024/01/112/ / https://zero-asia.biz/information/income_tax_01.html
- 信頼度: 高
- 内容: タイ居住者（183日以上滞在）は国内外源泉所得全てが課税対象（2024年以降）。非居住者はタイ国内源泉所得のみ課税。個人所得税率は0〜35%の累進課税。「183日未満滞在でノマドビザ取得」のニーズが一部増加しているが、ノマドビザ（LTRビザ等）でも183日超えるとタイ居住者と判定されることに注意。2024年改正はFAQ（歳入局発表）でも「タイ居住者の全国外所得送金は課税」と明確化。

## 情報14: 日本 出国税（国外転出時課税）の条件と対策
- 出典: https://www.smbc.co.jp/kojin/asset-management/wealth/kokusai-souzoku/004/ / https://mtng-tax.com/exit-tax/
- 信頼度: 高
- 内容: 出国税（国外転出時課税）の適用対象：①有価証券等の時価合計1億円以上、かつ②出国前10年以内に日本国内の住所・居所を5年超保有した居住者。対象資産：上場株式・投資信託・デリバティブ取引など（不動産は対象外）。未実現含み益に対して課税（最高55%）。対策：出国前に一部売却して納税資金を確保、または担保提供で納税猶予（5年間）を選択。10年以内に帰国かつ売却していない場合は課税取消し可能。

## 情報15: 日本 住民票を抜く最適タイミング
- 出典: https://bs.taxlawyer328.jp/blog/moving-abroad/p12862/ / https://biz.moneyforward.com/establish/basic/81699/
- 信頼度: 高
- 内容: 住民税は1月1日の住民票所在地で当年度課税が確定する（1月1日ルール）。翌年の住民税を回避するには、前年12月31日までに海外転出届を提出し住民票を除票する必要あり。手続きは出国1ヶ月前を目安に市区町村窓口または郵送で実施。住民票を残したまま出国すると「日本居住者」扱いが継続し住民税・国民健康保険料が課税され続ける可能性がある。株式・不動産売却益がある場合、出国日が1日違うだけで数十万〜数百万円の差になることも。

## 情報16: CFC税制（外国子会社合算税制）の適用条件とドバイ法人
- 出典: https://socpa.world/column/tax-haven-2025/ / https://maro-kaikei.co.jp/tax-haven-rules-japan/
- 信頼度: 高
- 内容: CFC税制（タックスヘイブン対策税制）は日本居住者・法人が50%超を保有する外国関係会社に適用。ドバイ（UAE）の法人税率9%は日本のトリガー税率（20%未満）を大幅に下回るため、原則として合算課税対象。適用回避の条件：①実体ある事業（従業員・事務所・実際の業務）を現地で持つ、②「経済活動基準」（事業基準・実体基準・管理支配基準・非関連者基準）を全て充足する。ペーパーカンパニー（実態なし）は合算課税を免れない。個人の場合、日本非居住者（183日以上海外滞在）となっていれば日本のCFC税制は原則不適用。
