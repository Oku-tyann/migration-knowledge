## 情報3: ドバイ フリーゾーン QFZP（法人税0%）資格条件（2025〜2026年版）

- 信頼度: 高
- 内容:
  - QFZP（Qualifying Free Zone Person）ステータスの4要件（全て同時に満たす必要あり）:

    4. **デミニミス要件**: 非適格収入が総収入の5%以下、かつAED 500万以下であること。
  - フリーゾーン登録だけでは0%税率は保証されない。ペーパーカンパニーの場合は9%課税対象。

## 情報7: タイ 183日ルール・2024年改正の実態

- 出典: https://www.ht-tax.or.jp/topics/thai2024-mochikomikazei-rules/ / https://www.yamada-partners.jp/global-report/th-report-240305 / https://kaipro.link/articles/foreign-sourced-income-brought-into-thailand/
- 信頼度: 高
- 内容:
  - **タイ居住者の定義**: タイ国内法では年間180日以上（租税条約では「183日」と記載）の滞在で居住者認定。
  - **2024年1月1日からの改正内容（実施済み）**:
    - 2023年9月通達（Departmental Instruction No. Por 161/2566）により、課税対象を拡大。
    - 従来: 「国外所得は所得が発生した年と同じ年内にタイに持ち込んだ場合のみ課税」
    - 改正後: 「2024年1月1日以降に稼得した国外所得は、タイに持ち込んだ年（何年後であっても）に課税」
    - 課税回避のため年またぎで送金するスキームは完全に封じられた。
    - 2024年1月1日より前に稼得した所得は対象外（経過措置）。
  - **LTRビザ（Long-Term Resident）保有者の例外**:
    - Wealthy Global Citizen・Wealthy Pensionerカテゴリーは2024年改正の影響を受けない特別優遇あり。
  - **日本人への実務的影響**:
    - タイ移住中に日本の金融資産（株式等）から得る所得・ドバイ法人からの配当等も持ち込み時に課税対象となる可能性。
    - 従来「タイ移住＝海外所得非課税」という認識は2024年以降は不正確。

## 情報9: 日本の出国税（国外転出時課税）の対象・条件・対策

- 出典: https://www.nta.go.jp/taxes/shiraberu/shinkoku/kokugai/01.htm / https://mac-g.co.jp/international-tax/column/102/ / https://shiodome.co.jp/column/7730/
- 信頼度: 高（国税庁公式含む）
- 内容:
  - **制度概要**: 平成27年（2015年）7月1日以後の国外転出者に適用。転出時に有価証券等の含み益に所得税（20.315%）を課税する制度。
  - **対象者の2要件（両方を満たす場合に課税）**:
    1. 出国時点で時価合計1億円以上の対象資産を保有。
    2. 転出までの過去10年以内に、5年超日本国内に居住していた。
  - **対象資産**:
    - 上場株式・非上場株式・投資信託・NISA口座内有価証券・米国株式・新株予約権等。
    - 2026年7月時点: 仮想通貨（暗号資産）は対象外。
  - **主な対策・選択肢**:
    1. **納税猶予制度**: 「納税管理人」を届け出ることで最大5年間（有価証券等の継続保有の場合は10年間）納税を猶予。猶予期間中に帰国すれば取消可能。担保の提供が必要。
    2. **事前売却・課税方法の選択**: 出国前に含み益のある資産を売却して確定させる（または損益通算）。
    3. **贈与・相続への波及**: 出国後5年以内に対象資産を日本居住者に贈与・相続した場合にも適用。
  - **申告期限**: 翌年3月15日（通常の確定申告と同じ）。帰国した場合は更正の請求で課税取り消し可。

## 情報10: CFC税制（タックスヘイブン対策）の仕組みと適用条件

- 出典: https://socpa.world/column/tax-haven-2025/ / https://www.jetro.go.jp/world/qa/04A-010814.html / https://maro-kaikei.co.jp/tax-haven-rules-japan/
- 信頼度: 高
- 内容:
  - **制度概要**: 外国関係会社（子会社等）の所得を、日本の親会社（株主）の所得に合算して課税する制度。外国法人を通じた日本課税の回避を防ぐ目的。
  - **適用トリガー税率**: 外国法人の租税負担割合が20%未満（ペーパーカンパニー等は30%未満）の場合に要件確認。
  - **UAEの位置づけ**: UAE法人税率9%は20%を下回るため、原則としてCFC税制の検討対象。ただしQFZP等で0%の場合も同様。
  - **適用除外（経済活動基準）**: 以下4基準を全て満たせば合算不要:
    1. 事業基準（主たる事業が株式保有・貸付・著作権使用料等の「受動的活動」でないこと）
    2. 実体基準（所在地国に事務所・設備等の実体があること）
    3. 管理支配基準（本店所在地国で管理・支配・運営されていること）
    4. 非関連者基準（収入の50%以上が関連者以外から）または所在地国基準
  - **ドバイ法人への実務**:
    - オンラインビジネスでドバイ拠点に実体を持たせることで経済活動基準クリアを目指すアプローチが多い。
    - 日本の株主が非居住者（日本の税法上）になれば、CFC税制の適用対象から外れる。

## 情報11: CFC税制 令和7年（2025年）度改正

- 出典: https://toc-tax.jp/2025/10/11/cfc/ / https://www.yamada-partners.jp/reform/r7/i02-review-of-the-foreign-subsidiary-tax-treatment-system
- 信頼度: 高
- 内容:
  - **合算時期の延長**: 令和7年度改正により、外国子会社の合算課税時期が「事業年度終了日の翌日から4ヶ月を超える日を含む内国法人の事業年度」に延長（旧：翌日から2ヶ月）。
  - **適用**: 親会社の2025年4月1日以後開始事業年度から適用。
  - **経過措置**: 2024年12月1日〜2025年1月31日に終了する外国子会社の事業年度については改正後の4ヶ月ルールを選択適用可。
  - **実務的意義**: 海外子会社の財務諸表作成・監査の時間的余裕が増える。日本親会社の申告期限の計算が変わるため、スケジュール管理の見直しが必要。
  - **個人株主への適用**: CFC税制は法人だけでなく個人（日本居住者）にも適用される。日本居住の個人が外国法人の株式の10%以上（直接・間接）を保有している場合に適用対象となりうる。

## 情報12: 日本住民票を抜く最適タイミング（1月1日ルール）

- 出典: https://bs.taxlawyer328.jp/blog/moving-abroad/p12862/ / https://biz.moneyforward.com/establish/basic/81699/ / https://www.ht-tax.or.jp/sozoku-guide/overseas-migration-tax
- 信頼度: 高
- 内容:
  - **住民税の1月1日ルール**: 住民税は「当該年の1月1日時点の住所地」で翌年度が課税される。
  - **翌年度住民税を非課税にするための要件**:
    - 前年の12月31日までに出国（海外転出）し、かつ市区町村に「海外転出届」を提出済みであること。
    - 例: 2026年度の住民税（2025年所得分）を回避するには、2024年12月31日までに転出届を出し出国。
  - **「国外転出届」の提出時期**: 出国1ヶ月前を目安に居住市区町村に提出（郵送も可）。
  - **非居住者認定（所得税）**: 日本の所得税法上の「非居住者」になるには「住所（生活の本拠）」と「居所（1年以上の見込み滞在）」が日本外にあることが要件。住民票の除票だけでは自動的に非居住者にはなれない（実態判断）。
  - **大きな含み益がある場合のタイミング戦略**:
    - 株式等売却益 → 非居住者確定後に売却で日本所得税回避可能（ただし出国税の1億円要件に注意）。
    - 不動産売却 → 売却タイミングと転出タイミングの組み合わせで課税額が大きく変動。
  - **帰国時の注意**: 1月2日以降に帰国・住民票再登録すれば、帰国年の住民税（前年所得分）は不課税。

## 情報13: マルタ 法人設立・実効税率の実態

- 出典: https://metiluxpartners.com/vol-1676 / https://www.eurovisa.jp/おすすめマルタ情報/マルタ法人設立-事業進出/ / https://rue.ee/jurisdictions/malta/
- 信頼度: 中
- 内容:
  - **名目法人税率**: 35%（EU圏では高い水準）
  - **実効税率の仕組み（完全帰属・税還付制度）**:
    - 法人が35%の法人税を納税後、株主（非居住者含む）が配当を受け取る際に「税還付申請」が可能。
    - 通常所得・受動所得: 株主は法人が納めた税の6/7を還付請求可能 → 実効税率**5%**
    - 印税・ロイヤルティ: 還付率5/7 → 実効税率**10%**
    - 持株会社所得: 還付率2/3 → 実効税率**約11.67%**
  - **EU加盟国のメリット**: EU単一市場へのアクセス、EU域内での法人信頼性、二重課税条約網。
  - **設立費用目安**:
    - 政府登録手数料: 最低資本金€1,165〜（Public Limitedは€46,588）
    - 年間維持コスト: ディレクター・法定監査費用等で年間€3,000〜8,000
    - 2025年3月以降は全書類オンライン提出必須・最短24時間での設立が可能。
  - **日本人への注意点**: 還付申請は株主名義で行うため、実務手続きが複雑。CFC税制との関係は実効5%が20%未満のため、日本居住者は合算課税リスクがある。移住前提での活用が現実的。

- 出典: https://atsoho.com/blog/georgia-hojin-setsuritsu-setsuzei / https://all-about.ge/tax_system/ / https://nomad-visas.com/country/georgia/ / https://kaigaijyu.net/ジョージアは日本からの移住先として向いているか/

- 内容:
  - **法人税制**:

    - スモールビジネス特例: 年間売上30,000〜500,000 GEL未満（約170万〜2,850万円相当）の場合、売上に対して**1%**のみ課税。

    - VAT: 18%（売上100,000 GEL超で登録義務）

## 情報15: ドバイ法人 銀行口座開設の難易度と対策（2025〜2026年版）

- 出典: https://socpa.world/column/bankopening/ / https://getfr33.com/dubai-account-opening/ / https://astravista-property.com/columns/dubai-golden-visa-company-bank-account-guide-2026
- 信頼度: 高
- 内容:
  - **全体的な難易度**: 高い（2024年以降にUAEがFATFグレーリストから除外されて以降、逆にKYC審査が厳格化）。
  - **審査期間**: 個人口座 3〜10営業日、法人口座 4〜12週間（複雑案件は3ヶ月超）。
  - **主要4行と特徴**:
    - Emirates NBD: 最大手。最低残高AED 50,000〜。法人設立実績が重要。
    - FAB（First Abu Dhabi Bank）: 国際取引に強い。最低残高AED 50,000〜。
    - Mashreq: 中小企業向け。日本人にも比較的オープン。
    - RAK Bank: コスト重視。最低残高AED 25,000〜。
  - **KYC審査の3軸**: ①UBO（実質的支配者）の特定 ②資金源（Source of Funds）の証明 ③ビジネス実態の証明。
  - **日本人の主な審査落ち原因**:
    - 資金源説明の不足・不明確
    - UAE居住ステータスが不明確（ビザ未取得状態）
    - ビジネスモデルが銀行側に理解されにくい（純オンライン事業）
  - **実務的推奨フロー**:
    1. まずWio Bank（デジタルバンク）やNeobaで開設・運用実績を積む
    2. 3〜6ヶ月の入出金履歴を作った上で主要4行へ申請
    3. DMCC・JAFZA等の信頼性の高いフリーゾーン法人であれば審査通過率が向上
  - **ゴールデンビザとの連動**: UAE居住者ビザ（または長期滞在ビザ）保有がスムーズな口座開設の前提条件として重要視されてきている。

## 情報16: 日UAE租税条約・二重課税防止の枠組み

- 出典: https://socpa.world/column/japanuaehikaku/ / https://nexpert-law.com/web3/dubai_houjinzei/
- 信頼度: 高
- 内容:
  - 日本とUAEの間には**租税条約（二重課税防止条約）が締結されている**（1993年発効）。
  - **主要規定**:
    - 配当: 源泉地国での限度税率10%（一定条件下）
    - 利子: 10%
    - 使用料（ロイヤルティ）: 10%
  - **実務的注意点**: 租税条約の恩恵を受けるためには、居住地国の申告・届出手続きが必要。条約上の「居住者」認定を受けていることが前提。
  - 日本のCFC税制と租税条約の関係: 租税条約の恩典は原則CFC合算課税に影響しない（CFC税制は租税回避対策のため条約上の保護外と解釈）。
