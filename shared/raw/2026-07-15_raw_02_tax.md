## 情報3: ドバイ フリーゾーン QFZP（法人税0%）適格条件（2026年版）

- 出典: https://astravista-property.com/columns/uae-corporate-tax-freezone-exemption-2026 / https://nexpert-law.com/web3/dubai_houjinzei/ / https://hls-global.jp/2025/10/10/uae経済的実体/
- 信頼度: 高
- 内容:
  - QFZP（Qualifying Free Zone Person）の4要件（全て同時に満たす必要あり）:
    1. **実質的経済的存在**: フリーゾーン内に事務所・従業員・事業経費が必要。
    2. **適格収入（Qualifying Income）**: 製造・加工・適格物品取引、関連会社向けヘッドオフィスサービス等。純オンラインビジネス（コンサル・仲介等）は非該当ケース多数。
    3. **監査済み財務諸表の作成**: アームズレングス原則への準拠が必須。
    4. **デミニミス要件**: 非適格収入が総収入の5%以下、かつAED 500万以下。
  - フリーゾーン登録だけでは0%税率は保証されない。実態のないペーパーカンパニーは9%課税対象。
  - 2026年12月31日まで小企業向け救済措置（de minimis簡便法）が継続。
  - 日本のCFC税制との関係: 日本人経営者が日本に居住したままだと、QFZPの有無に関わらず合算課税リスクあり。

## 情報7: タイ 183日ルール・2024年改正の実態（2026年確認）

- 出典: https://www.ht-tax.or.jp/topics/thai2024-mochikomikazei-rules/ / https://www.yamada-partners.jp/global-report/th-report-240305 / https://zero-asia.biz/information/income_tax_01.html
- 信頼度: 高
- 内容:
  - **タイ居住者の定義**: 年間180日以上（租税条約では「183日」と記載）の滞在で居住者認定。
  - **2024年1月1日からの改正内容（2026年現在も継続適用中）**:

    - タイ移住中に日本の株式配当・ドバイ法人からの配当等を持ち込んだ場合は課税対象となりうる。

## 情報9: 日本の出国税（国外転出時課税）の対象・条件・対策（2026年版）

- 出典: https://www.nta.go.jp/taxes/shiraberu/shinkoku/kokugai/01.htm / https://asean-jp.info/tax/japan-exit-tax-guide-2026/ / https://mac-g.co.jp/international-tax/column/102/
- 信頼度: 高（国税庁公式含む）
- 内容:
  - **制度概要**: 2015年7月1日以後の国外転出者に適用。転出時に有価証券等の含み益に所得税（20.315%）を課税。
  - **対象者の2要件（両方を満たす場合に課税）**:
    1. 出国時点で時価合計1億円以上の対象資産を保有。
    2. 転出までの過去10年以内に、5年超日本国内に居住していた。
  - **対象資産**: 上場株式・非上場株式・投資信託・NISA口座内有価証券・米国株式・新株予約権等。
  - **2026年7月時点**: 仮想通貨（暗号資産）は対象外。現金・不動産は直接の対象外。
  - **主な対策・選択肢**:
    1. **納税猶予制度**: 「納税管理人」届出で最大5年間（継続保有の場合最大10年間）猶予。担保提供が必要。
    2. **事前売却**: 出国前に含み益のある資産を売却して確定させる（または損益通算）。
    3. **贈与・相続への波及**: 出国後5年以内に対象資産を日本居住者に贈与・相続した場合にも適用。
  - **申告期限**: 翌年3月15日（通常の確定申告と同じ）。帰国した場合は更正の請求で課税取り消し可。
  - **回避方法まとめ**: ①含み益を1億円未満に抑える ②過去10年の日本居住期間を5年以下にする ③納税猶予制度を活用して帰国で取消

## 情報10: CFC税制（タックスヘイブン対策税制）の仕組みと適用条件

- 出典: https://socpa.world/column/tax-haven-2025/ / https://hls-global.jp/2025/10/10/uae経済的実体/ / https://www.ma-cp.com/about-ma/cfc-taxation/
- 信頼度: 高
- 内容:
  - **制度概要**: 外国関係会社（子会社等）の所得を、日本の親会社（株主）の所得に合算して課税する制度。外国法人を通じた日本課税の回避を防ぐ目的。
  - **適用トリガー税率**: 外国法人の租税負担割合が20%未満（ペーパーカンパニー等は30%未満）の場合に要件確認。UAEの9%(またはQFZP0%)は20%未満のため検討対象。
  - **経済活動基準（適用除外の4要件）**:
    1. 事業基準（主たる事業が株式保有・貸付・著作権使用料等の「受動的活動」でないこと）
    2. 実体基準（所在地国に事務所・設備等の実体があること）
    3. 管理支配基準（本店所在地国で管理・支配・運営されていること）
    4. 非関連者基準（収入の50%以上が関連者以外から）または所在地国基準
  - **ドバイ法人への実務**: オンラインビジネス単独では実体基準・管理支配基準のクリアが難しい。UAE拠点への実体移転（従業員・事務所・経営判断の実質的移転）が必要。
  - **個人株主への適用**: 日本居住の個人が外国法人株式の10%以上（直接・間接）を保有する場合に適用対象。日本の非居住者になれば対象外。

## 情報11: CFC税制 令和8年度（2026年度）改正の最新動向

- 出典: https://www.yamada-partners.jp/hubfs/zeikai/2026/i01.pdf / https://astravista-property.com/columns/uae-corporate-tax-freezone-exemption-2026
- 信頼度: 高
- 内容:
  - **解散特例の新設（令和8年度改正）**: 外国子会社が解散した場合でも、解散前2年間継続して経済活動基準を満たしていた場合、その後3年間は受動的所得のみが合算課税の対象となる特例が創設。
  - **適用開始**: 外国関係会社の2026年4月1日以後開始事業年度から。
  - **実務的意義**: 廃業・清算手続き中の外国子会社への過度な合算課税を緩和するもの。
  - **令和7年度改正の継続確認（合算時期延長）**: 親会社の2025年4月1日以後開始事業年度から、合算課税時期が「外国子会社の事業年度終了日の翌日から4ヶ月を超える日を含む内国法人の事業年度」に延長済み（旧：2ヶ月）。
  - **個人への適用**: CFC税制は個人（日本居住者）にも適用。日本居住者が外国法人の10%以上を直接・間接保有する場合が対象。

## 情報12: 日本住民票を抜く最適タイミング（1月1日ルール・2026年版）

- 出典: https://bs.taxlawyer328.jp/blog/moving-abroad/p12862/ / https://biz.moneyforward.com/establish/basic/81699/ / https://5ftraveler.com/movingabroad/residencetax/
- 信頼度: 高
- 内容:
  - **住民税の1月1日ルール**: 住民税は「当該年の1月1日時点の住所地」で課税される。住民票の所在地が基準（実態ではなく届出）。
  - **翌年度住民税を非課税にするための要件**:
    - 前年の12月31日までに出国し、市区町村に「海外転出届」（国外転出届）を提出済みであること。
    - 例: 2026年度住民税（2025年所得分）を回避するには2024年12月31日までに転出届を出し出国済であること。
  - **転出届の提出時期**: 出国1ヶ月前を目安に居住市区町村に直接または郵送で提出。
  - **非居住者認定（所得税）**: 日本の所得税法上の「非居住者」は「住所（生活の本拠）」と「居所（1年以上の見込み滞在）」が日本外にあることが実態で判断。住民票除票だけでは自動的に非居住者にならない。
  - **節税効果の試算**: 前年所得500万円の場合、住民税約50万円分を12月末出国で合法的に節約可能。1日の違いで数十〜数百万円の差が生じる。
  - **帰国時の注意**: 1月2日以降に住民票再登録すれば、帰国年の住民税（前年所得分）は不課税。

## 情報13: マルタ 法人設立・実効税率の実態（2025〜2026年版）

    - 印税・ロイヤルティ: 還付率5/7 → 実効税率**10%**
    - 持株会社所得: 還付率2/3 → 実効税率**約11.67%**
  - **2025年3月以降**: 全書類オンライン提出必須。申請から最短24時間での設立が可能。
  - **設立費用目安**:
