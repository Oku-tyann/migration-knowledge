## 情報6: 日本の出国税（国外転出時課税制度）— 適用条件と対策

- 出典: https://www.nta.go.jp/taxes/shiraberu/shinkoku/kokugai/01.htm / https://japan-dev.com/blog/exit-tax-japan / https://www.smbc.co.jp/kojin/asset-management/wealth/kokusai-souzoku/004/
- 信頼度: 高（国税庁公式含む）
- 内容: 適用対象の2条件（両方を満たす場合）— ①出国時点で対象資産の時価合計が1億円以上、②出国前10年以内に5年超日本国内に居住していた。対象資産（含み益に課税）は上場・非上場株式、投資信託、公社債、未決済デリバティブ取引等の有価証券（現金・不動産は対象外）。課税方式は出国時点の時価で全資産を売却したとみなして含み益に所得税・復興特別税を課税。納税猶予は税務代理人を選任し申告すれば最大5年（延長で10年）の猶予が可能。回避・軽減策として、出国前に対象資産を1億円未満に圧縮する（売却・贈与）、特定の就労ビザ滞在期間は5年カウントから除外（外国人には有利な規定）、帰国した場合（5年以内）は課税取消し。

## 情報7: 日本のCFC税制（外国子会社合算税制／タックスヘイブン対策税制）— ドバイ法人への適用

- 出典: https://www.jetro.go.jp/world/qa/04A-010814.html / https://taxfoundation.org/blog/japanese-cfc-rules-japan-tax/ / https://www.ey.com/en_gl/technical/tax-alerts/japan-2025-tax-reform-for-corporate-and-international-taxation
- 信頼度: 高
- 内容: 外国関係会社（CFC対象）の定義は日本の居住者・法人等が合計で50%超を直接・間接保有または実質支配する外国法人。納税義務者の範囲は当該外国関係会社の持株割合が10%以上の内国法人・居住者。「ペーパーカンパニー」「キャッシュボックス」カテゴリーの合算ラインは外国子会社の実効税率が27%未満（2024年4月1日以降開始する事業年度から、従来の30%から引下げ）の場合。経済活動基準（Economic Activity Test）で事業種別・固定施設・管理体制・独立企業間取引量の4基準を満たせば合算課税回避が可能。UAEドバイ法人の実効税率はフリーゾーンかつ適格収入の場合0%（または9%）で27%の閾値を下回るため原則としてCFC合算の対象。日本居住者のまま（住民票あり）でドバイ法人100%保有は、CFC税制によって法人の所得が日本の課税所得に合算される可能性が極めて高い。

## 情報8: 住民票を抜くタイミングと節税最適化

- 出典: https://biz.moneyforward.com/establish/basic/81699/ / https://bs.taxlawyer328.jp/blog/moving-abroad/p12862/ / https://www.zeiri4.com/c_3/q_100175/
- 信頼度: 高
- 内容: 住民税の1月1日ルール — 住民税は毎年1月1日時点の居住地に基づいて課税される。12月31日までに出国・住民票除票を完了させると翌年度の住民税（前年所得分）は発生しない。手続きは海外転出届を出国日の14日前〜当日に市区町村役場で提出。所得税の非居住者判定は原則として日本国内に住所がなくなった時点（非居住者認定）から国内源泉所得のみ課税。タイミング調整のポイントは①住民票除票と実際の出国を同時期に行うこと（ズレると脱税リスク）、②大きな利益（株式売却等）は住民票除票後に実現させる、③ドバイ法人への収益移転は非居住者認定後に開始する。注意点として住民票を抜いても実質的に日本に居住継続している場合（家族が日本在住・滞在日数が多い等）は税務署に「実質的居住者」と認定されるリスクあり。

## 情報9: タイの海外所得課税 — 2024年改正の実態

- 出典: https://www.yamada-partners.jp/global-report/th-report-240305 / https://www.hlbthai.com/new-rule-for-taxation-of-foreign-income/ / https://www.expattaxthailand.com/understanding-assessable-foreign-sourced-income-in-thailand/
- 信頼度: 高
- 内容: 改正の根拠は歳入局長命令 Por.161/2566（2023年9月15日付）、2024年1月1日施行。旧ルールは海外所得はタイへの「同一年内送金」が課税条件（翌年に送金すれば非課税の抜け穴が存在）。新ルール（2024〜）はタイ税務上の居住者（年間180日以上滞在）が海外所得をタイに送金した場合、所得発生年に関わらず送金した年に課税。適用開始前の所得（2023年以前）は2024年以降に送金しても非課税。DTA（租税条約）による免除・税額控除は引き続き適用可能。LTRビザ保有者は海外所得の課税が免除される特例あり。183日ルールの実態は正確には180日（「年間で180日以上」）で183日は他国との比較で混同されやすい。2025〜2026年の動向として外国源泉所得に対する課税緩和を検討中だが2026年6月時点で正式法制化は未確認。

## 情報11: エストニア e-Residency — 実際の税務と限界

- 出典: https://www.e-resident.gov.ee/taxes-in-estonia/ / https://capture.ee/estonian-company-tax/ / https://www.emta.ee/en/business-client/registration-business/non-residents-e-residents/tax-liabilities-companies
- 信頼度: 高（公式政府サイト含む）
- 内容: 法人税の基本構造は利益を社内留保している間は法人税0%で課税が発生するのは配当分配時のみ。配当税率（2025年1月〜）は22/78方式（旧20%→2025年から22%に引上げ）。例として株主へ€1,200の配当で会社が支払う税額 = €1,200 × 22÷78 = €338.46（実質的な税負担約22.2%）。2025年の変更点として「定期配当に対する14%優遇税率」と7%源泉税が廃止。e-Residencyカード取得費用は約€100〜120。会社設立費用は登記手数料€265 + 代行業者費用€200〜400+/月。最低資本金は€2,500（10年分割払い可）。コンタクトパーソン費用は年間€200〜400。税務上の重要な限界として、e-Residencyはエストニアの居住権や居住ビザを付与しない。法人の実質的な管理・支配が日本から行われている場合、日本の税務当局は当該法人を「日本居住法人」とみなす可能性がある。EU圏での銀行口座開設は年々困難化（Wise/Revolut等フィンテック活用が現実的）。日本のCFC税制の観点ではエストニア法人も「軽課税国法人」に該当する可能性があり注意が必要。

## 情報16: 日本人がドバイ法人でCFC税制を回避するための実務的条件

- 出典: https://www.grantthornton.jp/en/insights/news-letter/tax-bulletin/202604/ / https://www.chuointernational.jp/15222405434278
- 信頼度: 高
- 内容: CFC税制回避のためには以下すべてを充足する必要がある — ①実体要件（現地に固定施設・事務所を有すること）、②管理支配要件（事業の管理・支配・運営が現地で行われていること、日本からリモート管理では不可）、③所在地国基準（主要な事業活動が法人所在地国・UAEで行われていること）、④非関連者基準（売上の50%以上が関連者以外との取引から生じること）。2025年4月〜の変更として「ペーパーカンパニー」認定における実効税率の閾値が30%→27%に引き下げられた（UAEの9%はいずれにせよ閾値を大幅に下回る）。日本居住者のままドバイ法人100%保有の場合、法人の所得が個人の課税所得に合算される可能性が高く節税効果がほぼ消滅する。実務的推奨として専門の国際税務税理士による事前確認が不可欠。

## まとめ・実務上のポイント

日本人オンラインビジネス経営者が海外税務最適化を検討する際の最重要チェックリスト:

1. 住民票を抜く前に — 出国税（国外転出時課税）の対象（有価証券等の時価1億円以上）に該当しないか確認。該当する場合は事前の資産圧縮か専門家への相談が必須。
2. 住民票除票のタイミング — 実際の出国と同日に行うこと。1月1日を日本外で迎えることで翌年度の住民税が不要に。
3. CFC税制の適用リスク — 日本居住者（住民票あり）の状態でドバイ・エストニア・BVI等の低税率国に法人を設立しても、法人の所得が個人の課税所得に合算される可能性が高い。
4. ドバイ法人の費用感 — 最安はIFZA（年間AED 14,900≒約45万円）、最も信頼性が高いのはDMCC（初期投資AED 80〜90万円超）。
5. 実体ある節税のための国選択 — 実際に滞在・生活する意思があるならジョージア（1%）やシンガポール（実効5〜8%）、EU法人が必要ならエストニアまたはマルタが現実的。
6. タイの注意点 — 2024年以降、タイ居住者（180日超滞在）は海外所得をタイへ送金した年に課税される。旧来の「翌年送金非課税」スキームは使えない。
