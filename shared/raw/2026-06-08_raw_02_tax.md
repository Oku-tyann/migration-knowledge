# 税制・法人調査 — 2026-06-08

## 情報1: ドバイ フリーゾーン法人税 0% 条件（QFZP制度）
- 出典: https://www.khaleejtimes.com/business/finance/free-zone-corporate-tax-in-dubai-2025-beyond-the-0-myth
- 信頼度: 高
- 内容: UAE法人税は標準9%。フリーゾーン法人がQFZP（適格フリーゾーン法人）資格を取得すれば適格所得に0%が適用される。ただし条件あり：①コアとなる収益創出活動をフリーゾーン内で行う②十分な資産・従業員数・営業支出を維持③監査済みIFRS財務諸表の作成④非適格所得がAED500万または総収益の5%以下（De Minimis基準）。条件を1つでも満たさない場合、全所得に9%が適用され翌4年間も同様。フリーゾーン内他法人との取引収益が主な適格所得。

## 情報2: ドバイ フリーゾーン IFZA 設立費用・所要期間
- 出典: https://www.shuraa.com/ja/ifza-free-zone-company-setup/ / https://astravista-property.com/columns/dubai-free-zone-comparison-jafza-dmcc-ifza-dafz-2026
- 信頼度: 高
- 内容: IFZAはDubai Digital Park内に位置する低コスト志向フリーゾーン。設立費用はAED14,900〜（ゼロビザパッケージ）、ビザ1名含むとAED20,650（約83万円）程度。ビザ費用はAED3,750〜。ライセンス発行まで2〜3営業日。年間更新費用あり。設立全体の所要期間は約1ヶ月（ビザ取得含む、パスポートを現地預け2週間必要）。急ぎなら専門家サポートで1〜2週間も可能。

## 情報3: ドバイ フリーゾーン DMCC 設立費用比較
- 出典: https://www.shuraa.com/ja/dmcc-company-formation/ / https://getfr33.com/dubai-free-zone/
- 信頼度: 高
- 内容: DMCCはドバイ最大のフリーゾーンで信頼性が高いが高コスト。設立費用はAED20,000〜35,000（約80万〜140万円）が目安。IFZAと比較するとコストは約2倍。日本人向けエージェント経由の費用はさらに上乗せ。MEYDANも低コスト選択肢として人気。用途・業種によってIFZA、MEYDAN、DMCCを使い分けが有効。

## 情報4: シンガポール 法人税率・スタートアップ優遇制度
- 出典: https://statrys.com/sg/guides/tax-system-and-rates/corporate-tax-rate / https://www.iras.gov.sg/taxes/corporate-income-tax/basics-of-corporate-income-tax/corporate-income-tax-rate-rebates-and-tax-exemption-schemes
- 信頼度: 高
- 内容: シンガポール法人税は一律17%。ただしスタートアップ税額免除（SUTE）制度によりシンガポール設立法人の最初3年間は最大S$125,000の課税所得が免除され、実効税率は2〜4%まで低下。外国法人のシンガポール支店はSUTE対象外だが部分免除は利用可能。2026年度YAはCITリベート40%（最大S$30,000）。キャピタルゲイン非課税、配当源泉徴収税なし。

## 情報7: エストニア 法人税制（分配時課税方式）
- 出典: https://www.eestifirma.ee/lang/jp-corporate-income-tax-in-estonia/ / https://estonia-holic.com/run-business-in-estonia/
- 信頼度: 高
- 内容: エストニアは「分配時課税」方式を採用。利益を法人内に留保する限り法人税ゼロ。配当分配時に22%課税（2025年1月1日より20%から引き上げ）。計算式は純配当額×22/78（総額の約22%相当）。再投資・成長フェーズでは節税効果大。ただし日本居住者がエストニア法人から所得を得る場合、日本での個人所得税申告義務あり。タックスヘイブンとは異なり、EUの透明性基準に準拠。

## 情報8: ジョージア 法人税制・バーチャルゾーン制度
- 出典: https://all-about.ge/tax_system/ / https://lta.ge/en/article/company-formation-in-georgia-virtual-zone-entity / https://offshore-houjin.com/taxsavingknowledge/georgiaoffshore/
- 信頼度: 高
- 内容: ジョージアの通常法人税は15%フラット（分配時課税方式、留保利益は非課税）。IT・テクノロジー企業向け「バーチャルゾーン（VZ）」ステータスを取得すれば、ジョージア国外への収益に対して法人税0%・VAT0%。国内収益は15%課税。VZステータス取得にはオフィス賃貸・ジョージア人雇用が条件。設立費用は約100〜200ドルと格安。2017年よりエストニア型分配時課税を採用。

## 情報9: ジョージア スモールビジネスステータス（個人事業主向け）
- 出典: https://georgia.journey-coordinator.com/georgia-small-business-status/ / https://all-about.ge/tax_system/
- 信頼度: 中
- 内容: 個人事業主として登録し一定条件を満たすと年間売上に対して1%（条件次第で3%）の税率が適用されるスモールビジネスステータス制度あり。適用には年間183日以上のジョージア滞在（税務上の居住者）が必要。年間売上500,000ラリ（約2,700万円）未満が対象。デジタルノマドに人気だが、日本CFC税制・日本の課税義務との整合性要確認。

## 情報11: タイ 2024年改正 海外所得課税ルール
- 出典: https://www.yamada-partners.jp/global-report/th-report-240305 / https://bangken.com/column/2024/0305-3041289/ / https://pineconeblog.com/thailand-overseas-income-taxation/
- 信頼度: 高
- 内容: 2024年1月1日より、タイ居住者（年間180日以上滞在）が国外で得た所得をタイ国内に持ち込む場合、所得の発生年度に関わらず持ち込み年度にタイで個人所得税課税（累進5〜35%）。改正前は「同一暦年内の持ち込みのみ課税」だったが廃止。2024年1月1日以前発生の所得には適用なし。日タイ租税条約で一定の二重課税回避は可能。180日未満の非居住者は対象外。2025年以降の全世界課税化の議論も進行中（草案段階）。

## 情報12: タイ 183日ルールの現状と2025年以降の動向
- 出典: https://kaigaijin.jp/th/column/th-tax-guide / https://ilawasia.com/new-regulations-and-potential-laws-on-foreign-income-tax-in-thailand-jp/
- 信頼度: 中
- 内容: タイの税務上居住者基準は暦年180日以上滞在（実務上「183日」と表現されることが多い）。2024年改正後、居住者は2024年以降発生の国外所得をタイ持ち込み時に課税される。2025年には「タイ発生・タイ送金所得の全世界課税化」を含む個人所得税改正草案が議論中だが2026年6月時点で未施行。改正案では2025年所得を2027年以降送金した場合は累進課税対象。移住先としてのタイの税優位性は2024年改正で大きく低下した。

## 情報13: 日本 出国税（国外転出時課税）条件・対策
- 出典: https://shiodome.co.jp/column/7730/ / https://legacy.ne.jp/knowledge/before/fuyuusou/582-kokugaitenshutsu-kazeiseido-souzoku-zouyo-tekiyou-kaisetsu/ / https://www.nta.go.jp/taxes/shiraberu/shinkoku/kokugai/01.htm
- 信頼度: 高
- 内容: 2015年7月1日施行の国外転出時課税制度。適用条件：①出国時点で時価合計1億円以上の有価証券等（上場・非上場株式、投資信託、公社債、未決済デリバティブ等）を保有、かつ②出国前10年以内に5年超の日本国内居住歴がある者。含み益に対して所得税が課税される。対策：出国前に納税管理人届出を提出し確定申告に猶予申請を記載すれば最長5年（延長届出で10年）の納税猶予可能。出国後一定期間内に売却せず帰国した場合は課税取消し。資産1億円未満であれば適用なし。

## 情報14: 日本 住民票除票のタイミング（1月1日ルール）
- 出典: https://biz.moneyforward.com/establish/basic/81699/ / https://bs.taxlawyer328.jp/blog/moving-abroad/p12862/ / https://mailmate.jp/ja/blog/kaigaitenshutsu-juminzei
- 信頼度: 高
- 内容: 住民税は「1月1日時点」で日本に住所（住民登録）がある者に前年所得基準で課税される。翌年度の住民税を回避するには12月31日までに「海外転出届」を市区町村に提出し住民票を除票する必要がある。例：2026年12月31日以前に出国・除票完了→2027年1月1日時点で非居住者→2026年分所得に対する2027年度住民税は非課税。1月1日出国だと同日は日本に住所があるとみなされ課税対象になるため要注意。所得税（国税）の非居住者判定とは別概念。

## 情報15: CFC税制（タックスヘイブン対策税制）個人への適用条件
- 出典: https://koyano-cpa.gr.jp/nobiyo-kaikei/column/3941/ / https://suga-taxfirm.com/blogpost/cfc-04-resident-individual/ / https://www.supt.jp/blog/2891/
- 信頼度: 高
- 内容: タックスヘイブン対策税制は個人（居住者）にも適用される。対象：外国関係会社（内国法人および居住者等が合計50%超を支配する外国法人）の株式等を10%以上直接または間接保有する居住者個人。合算所得は雑所得として他の所得と合算・総合課税。累進税率（最高45%＋住民税10%）が適用されるため法人株主より税負担が重い。外国法人税の外国税額控除が個人には適用不可。外国関係会社の租税負担割合が20%以上の場合は原則適用除外。

## 情報16: CFC税制 ドバイ法人への適用と回避策
- 出典: https://socpa.world/column/tax-haven-2025/ / https://maro-kaikei.co.jp/tax-haven-rules-japan/
- 信頼度: 高
- 内容: UAE法人税率は標準9%（フリーゾーン適格所得0%）。日本のCFC税制のトリガー税率（20%未満）を大幅に下回るためドバイ法人は原則として合算課税対象に該当。回避策は2つ：①日本非居住者となる（住民票除票・海外転出）→CFC税制の適用対象外になる。②ドバイ法人に実体ある事業（従業員、オフィス、現地での意思決定等）を持たせ「経済活動基準」を満たす→会社単位の合算課税を回避できる可能性。ペーパーカンパニー状態では適用回避困難。
