## 【1】タイ LTRビザ（Long Term Resident Visa）概要

**出典:**
- https://asean-jp.info/thailand/thailand-ltr-visa-definitive-guide-2026/
- https://www.dlife.co.jp/blog/longtermvisa-thai/
- https://zero-asia.biz/information/visa_07.html

### 基本情報
- 正式名称: Long Term Residents Visa（長期居住者ビザ）
- 導入時期: 2022年9月
- 目的: 外国人富裕層・高度人材の誘致による消費・投資促進
- 有効期間: **10年**（5年ごとに更新手続きが必要）
- 種別: マルチプルリエントリービザ（何度でも出入国可能）
- 申請ポータル: `ltr.boi.go.th`

### 申請費用
- タイ国内取得: **50,000バーツ**（1人あたり）
- 申請審査期間: **約30日**
- 2025年3月〜: バンコク・One Bangkokに「TIESCワンストップサービスセンター」開設、申請プロセスが大幅簡素化

### 特典
| 特典 | 内容 |
|------|------|
| 扶養家族 | 配偶者＋20歳以下の子供（最大4名）まで同一ビザ発行可 |
| 個人所得税 | 優遇税率17%が適用可能 |
| 空港 | 国際空港ファストトラック利用可 |
| 90日レポート | 1年レポートに延長、再入国許可申請免除 |

### 取得条件（2025年2月改定後）
| 要件 | 内容 |
|------|------|
| 資産要件 | 最低**100万USD**以上の資産保有 |
| 年間所得要件 | **廃止**（2025年2月改定で撤廃） |
| 投資要件 | **50万USD以上**をタイ国債・タイ直接投資・タイ不動産のいずれかに投資 |
| 健康保険 | 5万USD以上の医療保険、または社会保険加入、または**10万USD以上の銀行預金**（12ヶ月以上）|

> **2025年の主要改定:** 従来要件だった「過去2年間の年間個人所得8万USD以上」が廃止。資産家・投資家へのハードルが大幅低下。

| 要件 | 内容 |
|------|------|
| 年収 | **8万USD以上** |
| 勤務形態 | 海外企業に対してリモートで勤務するデジタルノマド・リモートワーカー |
| 健康保険 | 5万USD以上の保険、または社保加入、または10万USD預金（12ヶ月以上）|

### 申請手続きの流れ
1. 企業情報・本人経歴などの資料を揃えて事前ガイダンスを受ける
2. 申請書類準備・提出
3. 審査期間（約30日）、事務局より許可証を取得
4. 新規入国者: 在住国のタイ大使館でビザ取得後入国
5. 在住者: 入国管理局にてLTRビザへの切り替え申請
6. ビザ取得後、労働許可証の新規発給または既存許可証から切り替え

## 【9】バンコク 180日滞在ルール — よくある誤解と正確な理解

**出典:**
- https://www.blue-assistance.co.th/wpvisa/overstay.php
- https://www.dlife.co.jp/blog/thailand-visa-newrules/
- https://x-bomberth.com/20251114visarunnomore/
- https://pattayalife.net/archives/74732

### 「180日ルール」の誤解
> **誤解:** タイには「年間180日まで滞在できる」という独立したルールがある
> **正確な理解:** 180日は「観光目的でのビザ申請却下の目安」であり、個別のビザ有効期間とは別概念

### 実際の滞在制限（観光目的）
| ビザ種別 | 滞在可能日数 |
|---------|------------|
| ノービザ入国 | 60日（空路）、延長+30日で最大90日 |
| 観光ビザ（TV） | 60日＋延長30日で最大90日 |
| 複数回の入国を組み合わせた上限 | 実務上120〜150日が安全圏 |

### オーバーステイの罰則
| 超過日数 | 罰則 |
|---------|------|
| 1日超過〜 | 500バーツ/日 |
| 罰金上限 | 20,000バーツ |
| 90日超過 | 超過日数に応じた**再入国禁止期間**が発動 |

### ビザラン規制強化（2025年11月〜）
- **2025年11月12日発表**: ビザランを「上限2回まで」と明確に制限
- 3回以上の繰り返しによる入国は、空港・国境問わず**入国拒否**の対象
- 2025年初頭〜同様パターンによる入国拒否者: 約**2,900人**
- 当局コメント: 「長期滞在希望者はビザランではなく、適切な長期滞在ビザを取得すべき」

## 【10】マレーシア DE Rantauノマドパス

**出典:**
- https://klnomad.com/malaysia-digital-nomad-visa/

- https://mishu.my/blog/visa/de-rantau-application/
- https://www.globalworkandtravel.com/resources/malaysia-digital-nomad-visa-the-complete-guide

### 概要
- 正式名称: DE Rantau Nomad Pass（Professional Visit Pass）
- 「Rantau」= マレー語で「探求する・旅する」の意
- 滞在期間: 最大**12ヶ月**（更新可能、合計最大2年）
- マルチプルエントリー権あり

### 対象職種（2024年6月以降拡大）
**Tech系:**
ソフトウェアエンジニア、バックエンドエンジニア、UX/UIデザイナー、クラウドコンピューティング、サイバーセキュリティ、ブロックチェーン、AI・機械学習、データ関連、デジタルマーケティング、デジタルコンテンツ制作

**Non-Tech系（2024年6月追加）:**
創業者/CEO、会計士、法律専門職、ライター等のプロフェッショナル管理職

### 収入要件
| カテゴリー | 最低年収 |
|-----------|---------|
| Tech系 | **USD 24,000/年** |
| Non-Tech系 | **USD 60,000/年** |

### 申請費用
- 単身（12ヶ月）: 約**RM 1,440以上**（約USD 315以上）
- 健康保険・保証金は別途
- **2025年5月1日以降**: 審査結果に関わらず申請費用は**返金不可**

### 申請処理期間
- **6〜8週間**

### 主な制約
- マレーシア国内企業での就労は不可（海外収入源のみ）
- ビザ有効範囲: **マレー半島のみ**（サバ州・サラワク州は別途観光ビザが必要）
- イスラエル国籍者は対象外

### 家族帯同
- 配偶者・扶養子女の帯同（審査あり）が可能

## 【17】補足: 長期滞在戦略の注意点

### タイにおける長期滞在の選択肢整理
1. **LTRビザ**: 条件厳しいが10年安定・就労可
2. **Thailand Privilege**: 費用高いが条件なし・就労不可
3. **ビザラン（ノービザ）**: 2025年11月より実質不可（2回上限）
4. **観光ビザ延長**: 年間累計120〜150日を安全圏とする

### ドバイの注意点
- 法人設立（フリーゾーン）経由でのビザ取得が最も一般的
- 2026年初頭から一部フリーゾーンはフリーランスビザ停止
- 個人所得税0%・法人税9%（2023年〜課税対象拡大）

### ジョージアの最新変更
- 「Remotely from Georgia」は終了済み
- 日本人はビザなし365日可能だが、2026年3月から就労権義務化
- 1%税率の個人事業主制度は継続中（最大USD 155,000まで）
