## 【1】タイ LTRビザ（Long Term Resident Visa）概要

**出典:**
- https://www.dlife.co.jp/blog/longtermvisa-thai/
- https://zero-asia.biz/information/visa_07.html
- https://asean-jp.info/thailand/thailand-ltr-visa-definitive-guide-2026/
- https://ltr.boi.go.th/

### 基本情報
- 正式名称: Long Term Residents Visa（長期居住者ビザ）
- 導入時期: 2022年9月
- 目的: 外国人富裕層・高度人材の誘致による消費・投資促進
- 有効期間: **10年**（5年ごとに更新手続きが必要）
- 種別: マルチプルリエントリービザ（何度でも出入国可能）
- 申請ポータル: `ltr.boi.go.th`

### 申請費用・手続き
- タイ国内取得: **50,000バーツ**（1人あたり、約20万円）
- 申請審査期間: **約30日**
- 2025年3月〜: バンコク・One Bangkokに「TIESCワンストップサービスセンター」開設、申請プロセスが大幅簡素化
- 新デジタルシステム導入により申請タイムライン短縮

### 主な特典
| 特典 | 内容 |
|------|------|
| 滞在期間 | 10年間（5年×2期） |
| 再入国許可 | マルチプル（無制限） |
| 90日レポート | 1年ごとの住所確認に簡略化 |
| 就労許可 | 一部カテゴリーで付与 |
| 税制優遇 | カテゴリー別（詳細は各項目参照） |

## 【2】タイ LTRビザ ― カテゴリー①：Wealthy Global Citizens（富裕層外国人）

**出典:**
- https://www.juslaws.com/articles/ltr-visa-wealthy-global-citizens-thailand
- https://thailand-ijyunavi.com/ltr-wealthy-global-citizens/

### 要件（2025年2月改定版）
- **資産要件**: 個人名義の世界資産 **100万USD以上**（金融商品・不動産が対象）
- **タイ国内投資**: 上記100万USDのうち **50万USD以上** をタイ国内に投資
- **年間所得テスト**: 2025年2月改定により**不要**（廃止）
- **医療保険**: タイ国内治療費 5万USD以上の保険 または 社会保険 または 銀行預金10万USD以上

### 税制優遇
- 外国源泉所得：**完全免除**（タイ標準累進税率最大35%と比較）

## 【7】ドバイ居住ビザ ― フリーランサー向け（グリーンビザ・フリーランスビザ）

### グリーンビザ（Green Visa）
- **対象**: フリーランス・自営業者
- **有効期間**: **5年間**
- **収入要件**: 過去2年間の自営業による年間収入 **360,000AED以上**（約1,455万円）
- **申請ルート**: メインランド政府機関またはフリーゾーン経由

### フリーランスビザ（Freelance Visa）
- **有効期間**: **1〜2年間**
- **対象職種**: WEB・映像デザイナー、プログラマー、教師、経営コンサルタント、フィットネスコーチ等
- **申請ルート**: フリーゾーンまたはメインランド

### リモートワークビザ（Digital Nomad Visa）
- **有効期間**: **1年間**
- **収入要件**: 月収 **5,000USD以上**（年60,000USD以上）
- 特徴: 海外雇用主の下でリモートワーク中の方向け

### 申請に必要な書類（グリーンビザ）
- 収入証明書または給与証明書
- 有効期間6ヶ月以上のパスポート・顔写真
- フリーランス許可証
- ポートフォリオ

## 【9】バンコク180日ルール ― よくある誤解と実際のオーバーステイ罰則

**出典:**
- https://www.blue-assistance.co.th/wpvisa/overstay.php
- https://thailandelite.info/overstay/
- https://chiangmai43.com/overstay/
- https://www.anzen.mofa.go.jp/od/ryojiMailDetail.html?keyCd=150298
- https://www.juslaws.com/criminal-litigation/overstay-thailand-blacklist

### 「年間180日ルール」について
- タイは観光目的の滞在を**年間180日まで**を目安としており、明らかに観光以上の滞在と判断された場合、ビザの再取得申請が却下されるリスクがある
- 「180日ルール」は法令上の明文規定ではなく、入国審査官による裁量運用の側面が強い
- 頻繁なビザランや長期連続滞在は「移住目的」と見なされ、入国拒否のリスクが高まる

### オーバーステイ（不法滞在）の罰則
| 超過日数 | 罰則 |
|----------|------|
| 1日ごと | 500バーツの罰金（上限20,000バーツ） |
| 90日超 | 出国後 **1年間** タイ入国禁止 |
| 1年超 | 出国後 **3年間** タイ入国禁止 |
| 3年超 | 出国後 **5年間** タイ入国禁止 |

### よくある誤解・注意点
- 「観光ビザを繰り返せばずっといられる」→ 入国官の裁量により拒否される場合あり
- 「空港出国時だけ罰金払えばいい」→ ブラックリスト登録・再入国禁止のリスク
- LTR/Thailand Privilegeなどの長期ビザ取得が、長期滞在の合法的な解決策として推奨される

- 有効期間: 最初 **12ヶ月**（更新可能）

### 所得要件
| カテゴリー | 年間収入要件（海外源泉） |
|------------|------------------------|

| 非テック系（経営職など） | **USD 60,000以上** |

※ 2024年6月改定: CEOや会計士、法律職、ライター等の専門管理職も対象に拡大

## 【11】ポルトガル D8ビザ（デジタルノマドビザ）

**出典:**
- https://wise.com/jp/blog/digital-nomad-visa-portugal
- https://hapiveri.com/blogs/nomad-life/portugal-living-abroad-with-a-digital-nomad-visa-visa-overview-and-requirements
- https://nomad-visas.com/country/portugal/
- https://rikiyaishizaki.com/portugal-d8-visa/d8-visa-requirements

### 基本情報
- 正式名称: D8ビザ（Visto D8 – Actividade Profissional à Distância）
- 導入: 2022年10月
- 最長滞在: **5年間**（初回4ヶ月 → 居住許可に切り替えで最大5年）

### 滞在・更新のフロー
1. **D8ビザ取得**: 有効期間4ヶ月（入国用）
2. ポルトガル到着後、**居住許可（Residence Permit）申請**: 初回2年間
3. **更新**: さらに3年間延長可能
4. 合計最大5年間の居住許可

### 最低滞在要件
- 最初の2年間の居住許可中: **16ヶ月以上**のポルトガル在住が必要

### 収入要件（2025年現在）
- **月額約3,040ユーロ以上**（ポルトガル最低賃金の4倍以上）
- 約年間36,480ユーロ（約580万円）以上

### 書類要件
- パスポートコピー・証明写真
- 収入証明（給与明細・フリーランス契約書など）
- 海外雇用または自営業証明
- 無犯罪証明書（アポスティーユ付き）
- 医療保険加入証明
- ポルトガル国内の住居証明（賃貸契約書など）
- 全書類: ポルトガル語または英語への翻訳・公証・アポスティーユが必要

## 【13】インドネシア E33G ビザ（デジタルノマドビザ / Remote Worker KITAS）

**出典:**
- https://www.thedigitalnomad.asia/digital-nomad-visa/indonesia-digital-nomad-visa/

### 基本情報
- 正式名称: E33G Remote Worker KITAS（KITAS = Limited Stay Permit）
- 有効期間: **1年間**（更新不可・新規申請が必要）
- 入国: マルチプルエントリー（複数回入国可）
- 入国期限: ビザ発行から **90日以内** に入国すること

### 収入・就労要件
- **年間収入**: **60,000 USD以上**（海外源泉収入のみ）
- **雇用形態**: 海外企業への**雇用（Employment）**が必要（フリーランス単独では不可）

### 財務要件
- 直近3ヶ月の貯蓄: **2,000 USD以上**

### パスポート要件
- 有効残存期間: **18ヶ月以上**
- 白紙ページ: **3ページ以上**

### 申請費用
| 費用項目 | 金額 |
|----------|------|
| 公式申請料 | 150 USD |
| KITAS登録料（入国時） | 165 USD |
| **公式合計** | **315 USD** |
| エージェント利用時の手数料 | 300〜600 USD追加 |
| **エージェント利用時合計** | **615〜920 USD** |

### 申請不可国籍
アフガニスタン、カメルーン、ギニア、イスラエル、コソボ、リベリア、ナイジェリア、北朝鮮、ソマリア

## 【16】ドバイ 居住ビザ取得ルートまとめ

**出典:**
- https://jcme-group.com/immigration_conditions/
- https://anna-advisors.com/usefulinformation/20250818-0930/
- https://www.starts-dubai.com/posts/42962296/

### 主なビザ取得ルート
1. **フリーゾーン法人設立**: 最も一般的。事業ライセンス取得と同時に居住ビザを申請。有効2〜10年。初年度コスト約120〜400万円。
2. **グリーンビザ（フリーランス・自営業）**: 年収360,000AED以上で5年間。収入証明が最大のハードル。
3. **フリーランスビザ**: 対象職種限定で1〜2年。フリーゾーン経由が一般的。
4. **リモートワークビザ**: 月収5,000USD以上の海外雇用者向け。1年間。
5. **不動産投資ビザ**: 75万AED以上（約3,000万円）の不動産購入で2〜10年の居住ビザ。
6. **ゴールデンビザ**: 200万AED以上の投資または高度人材で10年間。

## 【17】LTRビザ vs Thailand Privilege ― 選び方ガイド

**出典:**

### 選択基準
| 状況 | 推奨ビザ |
|------|---------|
| 資産・収入証明が難しい | Thailand Privilege |
| 就労許可が必要 | LTRビザ（WFT / Highly Skilled） |
| 外国源泉所得の税免除が必要 | LTRビザ（WGC / WP） |
| 短期で手軽に長期滞在したい | Thailand Privilege Bronze（5年・約260万円） |
| 富裕層で総合的な法的ステータスが欲しい | LTRビザ（WGC） |
| 50歳以上・不労所得あり | LTRビザ（WP）またはThailand Privilege |

### 税務比較
- **LTR WGC / WP**: 外国源泉所得 → **完全免除**
- **LTR WFT / Highly Skilled**: タイ国内勤務分 → **17%固定税率**
- **Thailand Privilege**: 独自の税制優遇なし（通常のタイ税法が適用）
