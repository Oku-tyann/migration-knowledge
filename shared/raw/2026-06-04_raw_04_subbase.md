## 概要

バンコク以外の海外居住・サブ拠点候補として12都市/地域（計15件以上の調査項目）を徹底調査。
各候補について「生活費・治安・ビザ・日本人コミュニティ・気候・インターネット環境」を網羅。

## 東南アジア系

### 1. マレーシア（クアラルンプール）

**生活費**
- 快適な一人暮らしの月額目安: RM 6,000〜10,000（約20〜33万円）
- 東京と比較して生活費全体で約42%安、家賃は50%以上安
- コンドミニアム家賃: RM 1,500〜3,000/月（約52,000〜105,000円）
- KL市内MRT乗り放題「My50パス」: RM 50（約1,650円）/月
- 東南アジア中でコスパ最強クラスと評価

**治安**
- 東南アジア基準では比較的良好
- 強盗発生率は日本の約25倍（荷物の置き忘れ・スマホ注意必須）
- 外出時・観光時も十分な警戒が必要

**ビザ**
- MM2H（マレーシア・マイ・セカンドホーム）プログラムあり
- デジタルノマド向けDE Rantauビザ（要最低収入証明）
- 観光ビザは90日間ビザなし滞在可能

**日本人コミュニティ**
- KLに在住日本人コミュニティ多数、日本語サービスも充実
- 日本食レストラン・日系スーパーも豊富

**気候**
- 熱帯気候、年間平均気温26〜30℃
- 雨季（10〜3月）と乾季（4〜9月）があるが年中高温多湿

**インターネット環境**
- KL市内は光回線・5G普及。平均速度100Mbps以上
- コワーキングスペースも充実（WORQ、Common Ground等）

**総評スコア（5点満点）**

### 2. ベトナム（ホーチミンシティ）

**生活費**
- 月額目安: $600〜1,200（格安〜快適ライフスタイル）
- 1Kアパート家賃: $300〜600/月（ホーチミン中心部）
- 食費: ローカル食なら月$100以下も可能
- 生活コスト最安クラスのノマド人気都市

**治安**
- 全体的に安全だがスリ・バイクひったくりには注意
- 外務省評価: レベル1（十分注意）

**ビザ**
- 日本人: ビザなし45日間滞在可能
- 長期滞在は就労ビザ（B1/B2）または学生ビザが実質的な選択肢
- デジタルノマドビザは未導入（2025年5月現在）
- 「10年ゴールデンビザ」構想: 2025年末〜2026年初のパイロット開始の可能性あり
- ビザランやビザ延長を繰り返す手法にはリスク

**日本人コミュニティ**
- 親日国で日本語学習者多数
- ホーチミンに数千人規模の在住日本人コミュニティ
- 日本食・日系企業が充実

**気候**
- 熱帯モンスーン気候、年間27〜35℃
- 乾季（11〜4月）と雨季（5〜10月）
- 暑さへの耐性が必要

**インターネット環境**
- 光回線は都市部で普及、平均速度50〜100Mbps
- カフェでのリモートワーク文化が発達

**総評スコア（5点満点）**

### 3. フィリピン（セブ）

**生活費**
- デジタルノマド月額目安: $800〜1,500（快適ライフ$1,500〜1,800）
- 1ベッドルームコンドミニアム: $300〜600/月（IT Park・Lahug・Banilad）
- 電気代: $60〜120/月（エアコン使用量による）
- 水道代: $10〜20/月
- セブは物価の安さとインフラのバランスが優秀

**治安**
- IT Park、ビジネスパーク等の主要エリアは比較的安全
- 地方・夜間は注意が必要
- 外務省: レベル1〜2（地域差あり）

**ビザ**
- デジタルノマドビザ: 2025年6月正式導入
  - 条件: 年収$24,000以上（遠隔勤務、フィリピン国外雇用主）
  - 初回12ヶ月、1回更新で最長24ヶ月
  - フィリピン税居住者にならない（所得税不要）
  - 要: 健康保険・犯罪歴なし証明
- 従来の観光ビザ: 最長36ヶ月まで延長可能

**日本人コミュニティ**
- セブに日本人学校・日本語学習塾が複数存在
- 英語学習目的の日本人短期滞在者も多い
- 日本食レストランあり（マニラ比べ少なめ）

**気候**
- 熱帯気候、年間25〜35℃
- 乾季（12〜5月）・雨季（6〜11月）
- 台風の影響を受けることがある（10〜12月）

**インターネット環境**
- PLDT/Globe光ファイバー: IT Park・Lahug等で100〜300Mbps
- 月額$30〜50
- IT Parkはコワーキングとカフェが充実

**総評スコア（5点満点）**
- コスパ: ⭐⭐⭐⭐⭐ / 治安: ⭐⭐⭐ / ビザ取りやすさ: ⭐⭐⭐⭐ / 日本人向け: ⭐⭐⭐

### 4. インドネシア（バリ）

**生活費**
- 快適な一人暮らし月額: $1,500〜2,500（エリアとライフスタイル次第）
- 1ベッドルームヴィラ/アパート（カングー/スミニャック）: IDR 8M〜20M（$500〜1,300）/月
- サヌール: IDR 5M〜10M（$315〜630）/月
- 2026年: カングーのレントは前年比18%上昇、敷金2ヶ月要求が増加

**治安**
- バリ島は比較的安全、外国人観光客・居住者多数
- バイクひったくりや詐欺に注意
- 外務省: レベル1

**ビザ**
- ビジタービザ（VOA/B211A）: 60日間、延長可で最長180日
- 第二の家ビザ（Second Home Visa）: 5〜10年長期滞在可能、要財産証明（$130,000+）
- 社会文化ビザ（B211A）での定期的なビザラン
- デジタルノマド向け「デジタルノマドビザ」は実質的に機能していないとの評価も

**日本人コミュニティ**
- バリ在住日本人は数千人規模、カングー・スミニャックに集中
- 日本食レストラン豊富、日本語対応サービスあり
- 日本人向けコミュニティグループ・イベント活発

**気候**
- 熱帯性気候、年間25〜32℃
- 乾季（4〜10月）・雨季（11〜3月）
- サーフィン・自然が豊かでノマドに人気

**インターネット環境**
- カングー・デンパサール等は光回線普及
- 速度は20〜100Mbps（エリアにより差が大きい）
- コワーキングスペース多数（Dojo、Outpost等）

**総評スコア（5点満点）**

### 5. カンボジア（プノンペン）

**生活費**
- 日本並みの生活: 月$1,000〜1,200程度
- ローカル寄りの生活: $500〜700も可
- コンドミニアム家賃: $400〜500/月（清潔な物件）
- 2人暮らし: 月$1,000〜1,200の実績あり
- 最低賃金（2025年）: $208/月
- 現地採用日本人月収目安: $800〜1,500

**治安**
- 日和見的窃盗・交通事故に注意
- 外務省: 全土レベル1（十分注意）
- プノンペン中心部の外国人エリアは比較的安全

**ビザ**
- Eビザ（業務ビザ）: 収入・学歴審査なし、毎年更新で長期滞在可能
- リタイアメントビザ（ERビザ）: 55歳以上対象、年約$300、最長12ヶ月
- ビザ取得のしやすさはカンボジアの最大の魅力の一つ
- 2025年9月: プノンペン新国際空港開港予定

**日本人コミュニティ**
- プノンペン在住日本人は数百〜1,000人規模
- 日本食レストラン・日本語学校あり
- 日本人向けコミュニティはKL・ホーチミン比では小規模

**気候**
- 熱帯モンスーン、年間25〜35℃
- 乾季（11〜4月）・雨季（5〜10月）

**インターネット環境**
- 都市部では光回線整備が進む
- 平均速度20〜50Mbps、コワーキングスペースは増加中

**総評スコア（5点満点）**
- コスパ: ⭐⭐⭐⭐⭐ / 治安: ⭐⭐⭐ / ビザ取りやすさ: ⭐⭐⭐⭐⭐ / 日本人向け: ⭐⭐⭐

## 地中海系

### 6. ポルトガル（リスボン）

**生活費**
- 1ベッドルームアパート（市内中心部）: €1,082/月（約17万円）
- 3ベッドルーム（中心部）: €2,647/月（約42万円）
- 郊外1BR: €500〜700/月
- 食費: €200〜300/月
- 生活費全体目安: €1,500〜2,500/月（単身）

**治安**
- ヨーロッパでも特に安全な国の一つ
- スリ・観光地での軽犯罪には注意
- 外務省: レベル1

**ビザ**
- デジタルノマドビザ（D8ビザ）: 年間最低収入€3,040×4倍（月€3,040以上）が条件
  - 初回1年、更新で最長2年
  - EU/EEA国籍以外が対象
- ゴールデンビザ: 投資移民向け（不動産等）
- パッシブインカムビザ（D7ビザ）: 年金・配当収入者向け

**日本人コミュニティ**
- 在住邦人数: 約737人（2021年）と少ない
- 日本語対応サービスは限定的
- 英語は都市部では通じる
- ヨーロッパ随一の「隠れた」日本人コミュニティが形成されつつある

**気候**
- 地中海性気候、年間15〜28℃
- 夏（6〜9月）は乾燥・晴天、冬（11〜2月）は温暖で雨が多い
- 欧州で最も日照時間が長い都市の一つ（年間3,000時間超）

**インターネット環境**
- 光回線普及率が高く、平均速度100〜200Mbps
- コワーキングスペース充実（Second Home、LACS等）

**総評スコア（5点満点）**

### 7. マルタ

**生活費**
- 単身月額: €1,700〜2,300（家賃含む）
- 家族(4人)月額: €4,000以上
- 1ベッドルームアパート（市内中心部）: €1,000〜1,500/月
- ゴゾ島: €500〜900/月（マルタ本島比40%安）
- 家賃除く生活費: €760以上/月
- 米国と比べ20〜30%安

**治安**
- EUで最も安全な国の一つ
- 犯罪率が非常に低い
- 英語が公用語で外国人が溶け込みやすい環境

**ビザ**
- ノマドレジデンスパーミット（デジタルノマドビザ）:
  - 最低年収: €42,000（月3,500€以上）※2025年10月時点
  - 最初の12ヶ月: 外国収入はマルタ所得税免除
  - 2年目以降: フラット10%税率
  - 更新: 年1回、最大4年まで
  - 滞在要件: 年間最低5ヶ月
  - 申請書類: 収入証明・居住証明・雇用契約・健康保険・犯罪歴証明
  - 処理期間: 通常30〜60営業日

**日本人コミュニティ**
- 日本人コミュニティは小規模
- 外国人居住者は人口の約30%（173,781人）と非常に多様な国際コミュニティ
- 英語が公用語のため言語面のストレスが少ない

**気候**
- 地中海性気候、年間12〜32℃
- 夏は暑く乾燥（7〜9月）、冬は温暖（1〜3月）
- 年間日照時間3,000時間超

**インターネット環境**
- 高速光回線が整備済み、平均速度100Mbps以上
- 小さな島ながらデジタルインフラは欧州水準

**総評スコア（5点満点）**
- コスパ: ⭐⭐⭐ / 治安: ⭐⭐⭐⭐⭐ / ビザ取りやすさ: ⭐⭐⭐ / 日本人向け: ⭐⭐

### 8. スペイン（バルセロナ/バレンシア）

**生活費**
- バルセロナ月額: €3,000〜4,500/月（快適ライフスタイル）
  - 1BR家賃: €1,400〜2,000/月（中心部）
- バレンシア月額: €2,200〜3,200/月（バルセロナ比30〜40%安）
  - 1BR家賃: €600〜1,000/月（中心部）
  - 1BR家賃: €500〜800/月（郊外）

**治安**
- バルセロナ: 観光地はスリ・置き引き多発（ラスランブラス周辺）
- バレンシア: バルセロナより比較的安全
- 外務省: レベル1

**ビザ**
- スペインデジタルノマドビザ（2023年導入）:
  - 最低月収: €2,850以上（スペイン最低賃金の200%）
  - 初回3年（任意更新可）、最終的にEU永住権取得ルートあり
  - 税制優遇: ベッカム法（特別駐在員税制）適用可能、非居住者課税（24%フラット）
  - 収入の15%はスペイン国外からでなければならない
- シェンゲン圏のため欧州全域への移動が容易

**日本人コミュニティ**
- バルセロナ: 在住日本人2,000〜3,000人規模、日本食・日本語学校あり
- バレンシア: 小規模ながら増加傾向

**気候**
- 地中海性気候
- バルセロナ: 年間10〜30℃、夏は暑く冬は温暖
- バレンシア: バルセロナより温暖、年間日照時間3,000時間

**インターネット環境**
- 欧州でも有数の高速通信インフラ
- 平均速度200〜300Mbps、コワーキングスペース多数

### 9. クロアチア

**生活費**
- 月額: €1,200〜2,000/月（手頃な欧州基準）
- 1BR家賃（市内中心部）: €600〜800/月
- ドブロブニクは観光地で高め
- 食費・交通費は西欧より30〜40%安い

**治安**
- 欧州でも安全な国の一つ
- 外務省: レベル1
- ドブロブニクは特に安全で観光整備が充実

**ビザ**
- デジタルノマドビザ（2021年導入、EU初期参入国）:
  - 2025年改定: 最長18ヶ月滞在可能に延長

  - 家族同伴: 1人追加ごとに平均賃金の10%（約€145）追加
  - 外国源泉収入のクロアチア所得税免除
  - 要: 健康保険（最低€30,000カバレッジ）：月€50〜150
  - 処理期間: 5〜12ヶ月（申請先・時期により差大）
- EUのシェンゲン圏加盟（2023年1月から）

**日本人コミュニティ**
- 在住日本人は非常に少数（数十〜数百人規模）
- 英語は都市部・観光地で通じる

**気候**

**インターネット環境**
- 都市部は高速回線整備済み
- 平均速度100Mbps以上、コワーキングスペースは拡大中

**ビザ**

  - 観光ビザでも海外向けリモートワーク可能に（制限なし）
  - フリーランサーや在宅勤務者に最適
- 専門職・永住権: スキルドビザ（Skills Shortage List）あり
- 専用デジタルノマドビザは未導入

**日本人コミュニティ**
- オークランド在住日本人: 10,000人以上（NZで最大）
- 日本語学校・日本食レストラン充実
- 日本人向けコミュニティ活動が活発

### 11. ジョージア（トビリシ）

- 1BR（郊外）: £321（約$405）/月平均
- 食費: $120〜200/月（地元食材）
- 外食: ヒンカリ/ハチャプリ 15〜25ラリ（$5〜9）

**ビザ**
- ジョージアは世界最大の無ビザ政策を持つ国の一つ

  - 条件: 月収$2,000以上または銀行残高$24,000以上

  - フリーランサー税率: 1%（最大の魅力）

## エージェントメモ・調査所感

- **最コスパ**: トビリシ（ジョージア）が月10〜15万円・ビザなし1年・1%フリーランサー税で圧倒的なコスパを誇る。ただし日本人コミュニティは小さく、日本語サービスなし。
- **バランス最良**: マレーシア（KL）が生活費・治安・日本人コミュニティ・ビザすべてにおいて高バランス。バンコクの代替拠点として最有力候補。
- **欧州での選択肢**: コストを抑えるならクロアチアまたはバレンシア。英語環境重視ならマルタ。
- **最安値帯（東南アジア）**: プノンペンはビザ取得が最も容易で生活費も最安。ただし治安・医療インフラはKL/ホーチミンに劣る。
- **気候的優位性**: メキシコシティの「永遠の春」（標高2,240m）は暑さが苦手な日本人に特に向いている。
- **2025〜2026年トレンド**: フィリピンのデジタルノマドビザ（2025年6月）、クロアチア18ヶ月延長（2026年）、ベトナムゴールデンビザ検討中など制度が急速に変化中。
