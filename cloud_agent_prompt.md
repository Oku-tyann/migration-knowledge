## タスク

以下の手順で海外移住情報を収集し、GitHubリポジトリにpushしてください。

### Step 1: 情報収集

以下のカテゴリから毎日ローテーションで検索する:

月曜: ビザ・在留資格（タイLTRビザ・タイランドエリート・ドバイ居住ビザ）
火曜: 税制・租税条約（183日ルール・タイ海外所得課税・ドバイ無税・出国税）
水曜: 生活コスト・住居（バンコク家賃相場・ドバイ家賃・生活費比較）
木曜: 法人・銀行口座（ドバイフリーゾーン・海外銀行口座・送金規制）
金曜: 治安・地政学リスク（タイ政治・UAE安定性・日本人コミュニティ）
土曜: 医療・保険（バンコク病院・海外健康保険・タイ医療費）
日曜: 総合（EU拠点候補・ポルトガル・マルタ・移住トレンド）

### Step 4: 保存・push

1. ファイルを `shared/raw/{DATE}_raw.md` として保存
2. git add, git commit -m "chore: {DATE} raw data collected", git push を実行

### 品質基準

- URLのない情報は掲載しない
- 公的機関（入国管理局・大使館・税務当局）の情報を優先
- 古い情報（2年以上前）には必ず「※{year}年時点の情報」と明記
- 「絶対節税できる」「今すぐ移住すべき」等の断定表現には⚠️を付ける
- 移住エージェント・仲介業者の記事はポジショントーク疑いをチェック
- 最低10件の情報を収集すること
